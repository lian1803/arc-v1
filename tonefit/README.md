# tonefit (가칭)

> 사장님 사진 5장만 주세요. 나머지는 그 톤으로 알아서 맞춰드립니다.

## 한 줄 요약
B2B 톤 통일 SaaS — 카페·뷰티샵 사장님이 인스타에 올리는 사진을 "그 가게의 톤"으로 자동 보정.

## 왜 만드나
- B2C 감성 보정 시장(VSCO·Lightroom·Evoto)은 포화.
- 그러나 **안목 없는 사장님 + 매일 올라가는 사진 + 그리드 톤 따로 노는 문제**는 아무도 안 풀고 있음.
- 사용자(=사장님)도, 만드는 우리도 "감성 안목"이 없어도 작동함 → 자기 기존 사진들끼리 통일하는 구조라서.

## 핵심 차별점
1. **B2B**: 일반 소비자 X, 자영업자 O.
2. **레퍼런스 = 본인 사진**: 외부 무드 학습 ❌, 사장님이 만족했던 자기 사진 5장이 곧 톤 정답.
3. **D:\work 마케팅팀 자산과 결합**: 카페·뷰티샵 납품팀 워크플로우에 끼워 넣을 수 있음.

## 현재 상태
- 아이디어 단계.
- 시장 리서치 1차 완료 (B2C 포화 / B2B 비어있음 / Pinterest 학습은 법적 리스크).
- **다음 액션: 사장님·인플루언서 5~10명 인터뷰로 수요 검증.**
