# Lian 1계정 채널 brand template 정확 스펙

## BLUF
Lian은 이미 일관된 brand template으로 50+ 썸네일 운영 중. 외부 corpus (60장) 패턴 ≠ Lian 채널 패턴.
- 외부 high-view 채널은 운동/후기 톤 / 다양한 색감
- Lian 채널은 **검정 배경 + 노랑 dominant + 의사 portrait** 통일
앞으로 Lian 채널용 자동화는 외부 패턴 X, **본 spec strict** 따라야 함.

## 정확 스펙

### Canvas
- 비율: 16:9 (실제 1280×720, preview ~605×340)

### Background
- **검정 단색** dominant (#000000 또는 매우 어두운 #0A0A0A)
- 빨강 background도 가끔 (e.g., 05번 "고지혈증 잡는 홍시" — 전체 빨강 #DC0000)
- **그라디언트 X**, 사진 X — 단색만

### 외곽 프레임
- **노랑 #FFEB1E** 외곽선 약 **5-8px** (얇음)
- 4면 모두 균일

### 인물 누끼
- 위치: **우측** 일관 (8/8 샘플)
- 사이즈: 캔버스 우측 35-45% 차지, 거의 풀 height
- 인물 종류: AI 생성 한국 의사 (남 60-70대 / 여 40-50대 mix)
- 표정: **진중함**, 직접 카메라 또는 살짝 옆 (위협/경고 톤)
- 복장: 흰 가운 또는 정장 + 안경 자주
- 누끼 fade: 좌측 가장자리 검정 배경과 자연스럽게 페이드 (rembg 후 좌쪽 추가 다크 그라디언트)

### 텍스트
- **위치**: 좌측 50-65% 영역
- **줄 수**: 3-4 줄 (가끔 5줄)
- **폰트**: 굵은 한국어 sans-serif (검은고딕 / Black Han Sans / 영원체)
- **크기**: 메인 ~80-100px, 보조 ~60-70px
- **색상**: 3색 only —
  - **노랑 #FFEB1E** = primary (60% — 위 줄 / 강조)
  - **흰 #FFFFFF** = body (30%)
  - **빨강 #FF0000** = strong emphasis (10% — "절대", "100%", "최악")
  - 가끔 형광 초록 #00FF00 도 (예: 04번 "가족이 췌장암")
- **외곽선**: 검정 4-6px (얇음 — 큰 글씨라 명도 OK)
- **bold 강조**: 빨강 글자에 **흰 외곽선 + 검정 그림자** 2단 stack도 가끔

### 우상단 채널뱃지
- 노랑 작은 박스 (~150×70px)
- 채널명 (예: "노년황금기") 검정 글씨
- 위치: 우상단 코너에서 안쪽 20-30px

### 상단 인용 (옵션)
- 흰 작은 글씨 (~40px) 인용 부호 포함
- 예: `"99%는 몰라서 후회합니다"` (반복 인용 / 채널 시그니처)
- 위치: 상단 가운데 또는 우측

### 카피 톤
- **위협 / 경고**: "절대 드시지 마세요!", "큰일 납니다", "직전입니다", "위험한 검사"
- **약속 / 결과**: "5분이면 기억력 유지됩니다!", "치매 없는 인생"
- **숫자 강조**: "5가지", "1500원", "100%", "5분"
- **단호한 어미**: "마세요", "됩니다", "납니다"

### 비교 — 내 V4 vs Lian 1계정

| 변수 | 내 V4 (외부 corpus) | Lian 1계정 (본 spec) |
|---|---|---|
| 배경 | 밝은 / 사진 / 다채 | **검정 단색** |
| 외곽 프레임 | 초록/빨강 10-12px 두꺼움 | **노랑 5-8px 얇음** |
| 텍스트 색상 | 5-7색 mix | **3색 (노랑/흰/빨강)** |
| 인물 톤 | 의사/트레이너/할머니 mix | **의사 only** |
| 카피 톤 | 후기 + 의사 mix | **위협 + 경고 only** |
| 큰 메달 | "300만 조회수" 리본 | **없음 (작은 채널뱃지만)** |
| 화살표 / quote bubble | 노랑 곡선 화살표 + 큰 quote | **화살표 X, quote 작은 흰 인용만** |

## 외부 정론 부합
- 외부 corpus (60장) 분석은 high-view 채널 전반 → 잘 맞지만 Lian 채널 특화 X
- Lian 채널은 본인이 이미 50+ 썸네일로 검증된 brand template
- **앞으로 Lian 채널 자동화 = 본 spec strict 따라야** (외부 corpus는 reference)
- 외부 corpus도 살아남는 변수가 있음 (예: 거대 텍스트 / 검정 외곽선) — 이건 spec에 포함됨
