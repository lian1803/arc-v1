Step 2

Google Cloud Platform이란? 구글이 제공하는 개발자 관리 도구입니다. 여기서 "내 서비스에 구글 로그인을 달겠다"고 등록하는 거예요.

Step 2 — 핵심
리다이렉트 URI가 뭐예요? — Supabase ↔ Google 연결 고리
⚠️
리다이렉트 URI 복사 방법
Supabase → Authentication → Google Provider → 아래 Callback URL의 Copy 버튼 클릭!
📋 URI 형태 예시
https://xxxx.supabase.co/auth/v1/callback
이걸 복사해서 Google Cloud에
그대로 붙여넣기하세요!
Step 3
Client ID를 받아서 Supabase에 넣기
🎯 핵심 정리
결국 Supabase와 Google Cloud가
서로를 인증하는 과정입니다.

Google → Supabase:
"이 Client ID로 나를 확인해"

Supabase → Google:
"로그인 끝나면 이 주소로 보내줘"
(= Redirect URI)

Step 5
Supabase API 키를 코드에 넣기
AI가 만든 코드에는 빈칸 2개가 있어요.
Supabase에서 값을 복사해서 넣어주면 됩니다!
script.js
// TODO: 아래 값들을 실제 Supabase 정보로 바꾸세요
const SUPABASE_URL = '여러분의_URL'
// 예: https://xxx.supabase.co
const SUPABASE_KEY = '여러분의_키'
// 예: eyJhbGciOiJIUzI1NiIs...
supabase.com → Settings → API
📍 Supabase에서 복사할 것 2가지
① Project URL
https://xxx.supabase.co
→ script.js의 SUPABASE_URL에 붙여넣기
② anon public Key
eyJhbGciOiJIUz... (긴 문자열)
→ script.js의 SUPABASE_KEY에 붙여넣기
💾
붙여넣기 후 반드시 저장!
Windows: Ctrl+S / Mac: Cmd+S
