## ICP — 진짜 사용자 1명 (가상 인물, 디테일까지)
- 이름: 정수민 (가명) / 나이: 38세 / 성별: 여
- 직업: 동네 1인 미용실 원장 ("수민헤어", 서울 강서구 화곡동, 1층 6.5평 매장, 임대료 보증금 2,000 / 월세 90)
- 경력: 미용 13년차. 강남 프랜차이즈 헤어디자이너 7년 → 동네 분점 부원장 2년 → 5년 전 화곡 독립 개업
- 매출/구조: 월 매출 약 700만원, 재료비/임대료/공과금 제외 후 손에 남는 돈 월 380-420만원. 주 5.5일 영업 (월·수~토 풀, 일 단축, 화 휴무)
- 가족: 남편(직장인 41세), 아이 2명 (초3 딸, 초1 아들). 워킹맘. 친정엄마 도움 받음.
- 고객 구성: 단골 70% / 신규 30%. 단골의 70%가 매장 반경 1km 이내 거주 (걸어서 옴)
- 시술 구성: 컷 35% / 펌 30% / 염색 25% / 케어 10%. 객단가 평균 4.2만원
- 일상: 오전 10시 매장 오픈 → 21시 마감. 점심은 컵라면 또는 김밥(손님 빠진 14-15시 사이). 화요일은 아이들 학교 끝나는 시간 맞춰 학원 픽업 동행.
- IT 환경: 카카오톡 1:1 채팅으로 예약 받음 (단골은 친구추가, 신규는 매장 QR로 추가). 종이 다이어리에 손글씨 예약. 인스타 계정 있지만 작년 9월 이후 미운영. 노트북 X, 매장에 PC X. 폰은 갤럭시.
- 돈에 대한 태도: 월 1만원짜리 결제도 "이게 진짜 도움 되나?" 한 달 고민. 다만 "확실한 매출 손실 막아주는 것"엔 즉시 지불 (예: 노쇼 보증제 잠깐 시도한 적 있음).

## Pain — Top 3 (실제 정수민 표현 그대로)
1. **"그 손님 지난번에 뭐 했더라... 또 까먹었네"** — 단골 시술 이력 기억 X. 들어오면 "어, 안녕하세요!" 하고 차트 뒤지면서 "지난번 펌 어떠셨어요?" 묻고 싶지만 못 함. 손님이 "저번에 했던 그거 한 번 더 해주세요" 하면 머릿속이 하얘짐. 신뢰 깎이는 순간.
2. **"또 노쇼야. 30분 비었어"** — 카톡 예약 받으면 약속 직전 까먹는 손님 30%. 펌 2시간 잡아둔 자리 비면 매출 8만원 직격 손해. 카톡으로 "내일 11시 맞으세요?" 일일이 보내자니 50명 단골 전부 못 챙김.
3. **"다음 약속 잡아드려야 하는데 늘 깜빡"** — 컷 끝나고 "4주 뒤쯤 다시 오시면 좋아요" 한마디 못 함. 손님은 시간 지나면 "어디 가지~" 하다 다른 미용실 가버림. 단골이 자연 소실되는 걸 그냥 봄.

(베이스: 실제 화곡/목동 동네 미용실 1인 사장 인터뷰 풍 — 본 PLAN의 검증 가설 1로 검증할 가정)

## Solution — 카톡으로 단골카드 자동 (이름 가안: "수민헤어 같은 1인 원장용 카톡봇")
- 핵심 기능 3개만 (UI 추가 X):
  1. **예약 30초 입력** — 카톡으로 단골이 "다음주 화 11시 가능?" 보내면 정수민이 평소처럼 "네 가능해요" 답하고, 앱 열어 "예약 추가" 1탭 → 단골 이름 선택 → 시간 입력 → 저장 (총 30초). 자동 파싱 X (Reach Gate 2026-04-29: 카카오 정책상 1:1 자동 캡처 불가능 — A안 채택).
  2. **시술 후 1탭 메모** — 시술 끝나고 폰 열어 손님 이름 누르고 "오늘: 매직 + 회색 톤다운, 다음 4주 후 컷" 한 줄 적으면 저장. 다음 방문 시 입장 알림과 함께 "지난번: 매직 회색 / 마지막 방문 28일 전" 1줄 표시.
  3. **알림톡 자동 발송** — (a) 예약 24h 전 "내일 11시 수민헤어입니다 :)" 알림톡 자동. (b) 시술 4주차 "수민헤어입니다. 컷 시기네요~" 알림톡 자동. 카카오 비즈채널 가입은 Lian이 발품으로 도움 (Do Things That Don't Scale, baemin 학습 #1 직접 차용).
- 차별점:
  - 매장에 PC 둘 자리 없음 → 모든 게 폰 1개로. 대시보드 X.
  - 정수민 손 = "예약 추가" / "메모" 2버튼만. 비즈채널 운영은 ARC가 (Lian이 가입 발품으로 도와준 후 자동 발송).
  - 알림톡 비용 = 건당 7-15원 (월 100건 = 약 1,500원). Pro 19,000원에 포함.
- baemin 차용 흔적: "동네 음식점 정보 흩어짐 → 한 곳" 패턴을 "단골 정보 원장 머릿속 흩어짐 → 앱 1탭"으로 미러. Reach Gate 1차 적용으로 BYOK 동형 패턴 회피 (invoice-snap 실패 학습).

## 가격 모델
- **무료 (영원히)**: 손님 50명 제한 + 자동 카톡 알림 X. 손님 카드만.
- **Pro 월 19,000원**: 손님 무제한 + 자동 카톡 알림 (예약 24h 전 + 시술 후 4주차).
- **첫 1년 Pro 무료** (PMF 검증 단계 — 50명 베타 한정).
- 가격 근거 (baemin 결로 비교):
  - 정수민 객단가 4.2만원. 노쇼 1번이 8만원 손실. 월 19,000원 = 노쇼 1번의 4분의 1만 막아도 본전.
  - "한 달 10만원짜리 매장 결제 시스템" 동네 미용실 못 씀. "월 1만원대"가 정수민 머릿속 결제 ceiling. 19,000원은 그 위 아슬아슬.
- 광고/수수료 모델 X — 자영업자에게 "수수료 떼이는" 신호는 즉시 거부감 (배민 점주 이탈 트렌드 직계 학습).
- 매출 시뮬: Pro 100명 = 월 190만원. 1,000명 = 월 1,900만원. 동네 미용실 한국 약 11만개 (통계청 2022) → 0.1% 침투만으로 110개 = 월 209만원.

## 검증 가설 3개 (각각 falsifiable)
1. **수락률 가설**: 화곡동 30곳 발품 → 30%(9곳) 이상이 "다이어리 사진 무료 디지털화" 제안 수락한다.
   - 거짓이면: 무료 제안조차 거절 = "단골 기억 못함"이 정수민 머릿속 Top 3 페인이 아니거나, 외부인 매장 출입 자체가 부담. → 다른 세그먼트(동네 네일샵/속눈썹) 또는 진입 방식(카톡 광고 → 자발 신청) 재검토.
2. **자발 사용 가설**: 디지털화 받은 9곳 중 50%(4-5곳) 이상이 4주 안에 자발적으로 손님 카드 1회 이상 열어본다.
   - 거짓이면: 일회성 호기심뿐 지속 가치 X = 손님 카드 자체가 페인 해소 안 함. → solution 재설계 (손님 카드 X, 노쇼 알림 단독 또는 매출 분석 단독).
3. **유료 전환 가설**: 4주 후 자발 사용자 4-5곳 중 30%(1-2곳) 이상이 "1년 무료 끝나면 월 19,000원 내고 쓰겠다" 명시 답변한다.
   - 거짓이면: 가격 민감도 ceiling 1만원 이하 = 19,000원이 너무 비쌈. → 가격 재산정 (9,900원) 또는 매출 모델 변경 (재료 공동구매 마진 등).
