## 2. 앵커링 구조

### 왜 C(95만)를 먼저 보여주는가
- 첫 제안: A~C 전체 메뉴 제시 → C 가격(95만) 앵커 설정
- B(56만)가 "합리적인 선택"으로 인식됨 (95만 대비 60% 수준)
- A(38만)가 "이건 진짜 부담 없네"로 인식됨

### 제안 순서 (현장 영업)
```
1단계: C(95만) 먼저 설명 → "이게 전체 패키지예요"
2단계: "근데 처음이시면 B나 A부터 시작하는 분들이 많아요"
3단계: "이 업체엔 B가 딱 맞을 것 같아요" → 업체 상황 기반 개인화 제안
```

## 4. 가격 협상 대응

### "너무 비싸요" 대응 시나리오

| 상황 | 대응 |
|------|------|
| B(56만)가 비싸다 | "그럼 A(38만원) 먼저 해보세요. 3개월 해보고 효과 보이면 B 시작하는 분들 많아요" |
| C(95만)가 비싸다 | "광고비는 사장님이 직접 결제라서 대행료만 95만원이에요. 광고비 30만원이면 총 125만원인데 신규 고객 10~20명 유입되면 BEP 나와요" |
| A(38만)가 비싸다 | "한 달에 38만원이면 직접 하실 때 시간 대비 비용이에요. 플레이스 관리 혼자 하시면 한 달에 15~20시간 걸려요" |

### 절대 하면 안 되는 것
- 가격 즉시 깎아주기 → 전체 가격 체계 무너짐
- "일단 무료로 해드릴게요" → 이후 유료 전환 불가
- 패키지 구성 즉석 변경 → 원가 계산 불가

## 5. 계약 구조 설계

### 결제 방식
- 월정액 (A, B, C): 첫 달 100% 선불 + 이후 매월 초 청구
- 최소 계약: 3개월 → 위약금: 잔여 개월 × 50%

### 광고비 처리
- C 패키지 광고비: 고객 카드 또는 계좌로 직접 결제 (투명성)
- 우리가 광고비 받아서 집행하는 구조 = 분쟁 원인 → 절대 금지

### 계약서 필수 조항
1. 성과 보장 없음 명시 (순위, 매출 증가 보장 금지)
2. 리워드 트래픽 사용 시: 플레이스 페널티 발생 시 에이전시 귀책 면책
3. 영수증 리뷰 상품 사용 시: 네이버 정책 위반 적발 시 광고주 책임 명시
4. 블로그 체험단: 뒷광고 표시 의무 위반 시 광고주 책임
5. 해지: 최소 계약 기간 내 해지 시 잔여 월 × 50% 위약금

## 6. 성장 경로 (고객 LTV 설계)

```
무료 진단
    ↓ (전환율 목표 20%)
A. 플레이스 케어 (월 38만, 3개월)
    ↓ (2개월차 업셀 제안, 전환율 목표 40%)
B. 바이럴 성장 (월 56만, 3개월)
    ↓ (3개월 후 제안, 전환율 목표 30%)
C. 광고 풀케어 (월 95만, 6개월+)
```

**고객 1인 평균 LTV (12개월 시나리오)**
- A × 3개월 → B × 3개월 → C × 6개월
- = (380,000 × 3) + (560,000 × 3) + (950,000 × 6)
- = 1,140,000 + 1,680,000 + 5,700,000
- = **약 852만원/년 (1인)**
