# 데이터 분석 서비스 — 리드 수집 가이드

> 어디서 타겟을 찾고, 어떤 업체부터 연락해야 계약률이 높은지 정리한 가이드.
> 이커머스/D2C 브랜드 대표 및 마케터가 타겟이다.

---

## 1. 리드 자격 기준 (이 업체가 우리 타겟)

아래 조건 중 2개 이상 해당하면 적극 공략한다.

| 기준 | 확인 방법 | 이유 |
|------|----------|------|
| 월 광고비 300만원 이상 집행 중 | Meta 광고 라이브러리에서 집행 광고 확인 | 광고비 있어야 데이터 니즈 생김 |
| 자체 쇼핑몰 or 자체 사이트 보유 | Cafe24, 아임웹, Shopify, 자체 도메인 | GA4/GTM 세팅 대상이 됨 |
| 메타 or 구글 광고 집행 중 | 광고 라이브러리, 구글 투명성 센터 | 픽셀/CAPI 니즈 확실 |
| GA4 or 픽셀 세팅 상태 불확실 | Tag Assistant, Pixel Helper로 사전 확인 | 세팅 개선 여지 있음 |
| 브랜드 인스타 운영 중 (팔로워 1천 이상) | 인스타 검색 | D2C 브랜드로 직접 판매 중 |

### 즉시 제외 기준 (연락 낭비)

- 월 광고비 100만원 미만 (스타터 비용도 부담스러운 규모)
- 스마트스토어만 운영하고 자체 사이트 없음 (GA4 세팅 대상 없음)
- 오프라인 전용 사업자 (온라인 전환 데이터 니즈 없음)
- 이미 대형 대행사 계약 중이고 데이터팀 내부에 있음

## 2. 타겟 소스별 리드 수집 방법

### 소스 1: 스마트스토어 상위 노출 셀러

**왜 여기인가:** 스마트스토어 상위 노출 셀러는 이미 광고비를 집행하고 있고, 매출이 있는 업체다. 근데 자체 쇼핑몰로 확장하거나 메타 광고를 병행하는 경우, 트래킹 세팅이 엉성하게 되어 있는 경우가 많다.

**식별 방법:**
1. 네이버 쇼핑에서 주요 카테고리 검색 (예: "여성의류", "건강식품", "반려동물용품")
2. 상위 노출 10~20개 제품 클릭 → 판매처가 자체 브랜드 스토어인지 확인
3. 리뷰 1천 개 이상이고 월 판매량 50개 이상 → 규모 있는 셀러
4. 스토어 홈 들어가서 공식 사이트 링크 확인 → 자체 쇼핑몰 있으면 우선 타겟

**확인할 것:**
- 자체 사이트 있는지 (Cafe24, 아임웹, 자체 도메인)
- 메타/구글 광고 집행 중인지 (광고 라이브러리 검색)
- GA4 세팅 여부 (Tag Assistant)

---

### 소스 2: Cafe24 / 아임웹 운영 쇼핑몰

**왜 여기인가:** 이 플랫폼 쇼핑몰은 GA4 세팅이 수동이라 빠진 경우가 많다. 메타 CAPI 연동도 직접 해야 해서 미세팅 비율이 높다. 세팅 니즈가 가장 확실한 집단이다.

**식별 방법:**
1. 구글 검색: `site:cafe24.com [카테고리키워드]` 또는 구글 쇼핑 광고 노출 업체
2. 아임웹 스토어 디렉토리: imweb.me에서 카테고리별 노출 스토어
3. 인스타 쇼핑 링크 클릭해서 Cafe24/아임웹으로 이어지는 계정
4. 네이버 광고 → 랜딩 URL이 Cafe24 도메인인 경우

**바로 확인할 것:**
- Tag Assistant로 GA4 세팅 여부
- Pixel Helper로 픽셀 이벤트 세팅 상태
- 실제로 광고를 돌리고 있는지 (광고 라이브러리)

---

### 소스 3: 인스타 D2C 브랜드 (팔로워 5천~5만)

**왜 여기인가:** 팔로워 5천~5만 구간은 광고비 집행을 막 시작했거나 급격히 늘리는 시기다. 매출은 나오고 있는데 데이터 인프라가 없어서 광고 효율을 모르는 상태가 많다.

**식별 방법:**
1. 인스타 검색: 카테고리 키워드 + "쇼핑 링크", "구매 링크", "링크인바이오"
2. 팔로워 5천~5만, 최근 게시물 인게이지먼트 1% 이상
3. 프로필에 외부 링크 있고, 자체 도메인 쇼핑몰로 이어지는 계정
4. 메타 광고 집행 중인지 광고 라이브러리에서 확인 (계정명 검색)

**우선 타겟 시그널:**
- 프로필에 "D2C", "공식몰", "직구매" 등 표현
- 제품 태그 많고 릴스 광고 집행 중
- 링크트리나 숍더룩 연결된 계정

---

### 소스 4: 화해 / 올리브영 입점 뷰티 브랜드

**왜 여기인가:** 화해 입점 브랜드는 자체몰 + 메타 광고 병행하는 경우가 많다. 뷰티 카테고리는 재구매율이 높아서 CRM + 트래킹 연동 니즈가 강하다. 객단가도 높아서 데이터 분석 ROI가 크다.

**식별 방법:**
1. hwahae.com에서 브랜드별 필터 → 입점 브랜드 목록 확인
2. 브랜드 검색해서 공식 사이트 있는지 확인
3. 올리브영 온라인몰 인기 상품 → 브랜드 공식 사이트 유무 확인
4. 해당 브랜드 인스타 검색 → 광고 집행 여부 확인

**특히 좋은 타겟:**
- 올리브영 입점이면서 자체몰도 운영 중
- 인스타 팔로워 1만 이상이고 광고 집행 중
- 제품 리뷰 300개 이상 (매출 있다는 증거)

---

### 소스 5: 쿠팡 로켓그로스 파트너

**왜 여기인가:** 로켓그로스는 쿠팡 외부에서 자체 광고도 병행하는 경우가 많다. 쿠팡 내 데이터는 제한적이라 외부 광고 효율을 따로 측정해야 하는 니즈가 있다.

**식별 방법:**
1. 쿠팡에서 "로켓그로스" 뱃지 달린 상품 검색 → 브랜드명 확인
2. 해당 브랜드 구글 검색 → 자체 사이트 있는지 확인
3. 인스타/페북 광고 집행 여부 확인

**주의:** 로켓배송(쿠팡 직매입)은 셀러가 아니라 쿠팡이 팔아서 타겟 아님. 로켓그로스(셀러가 직접 관리)만 타겟.

## 3. 아웃리치 우선순위 (계약률 높은 순서)

### 1순위: 광고 집행 중 + GA4/CAPI 미세팅 업체

이미 광고비를 태우고 있는데 데이터 인프라가 없는 상태. Pain이 이미 존재하고, 설득 비용이 낮다.

접근 방식: 사전 진단 결과를 들고 간다.
```
"사이트 미리 보니까 GA4는 있는데 이벤트가 안 잡히더라고요. 
지금 집행 중인 메타 광고 전환이 실제 매출이랑 얼마나 일치하는지 확인해드릴게요."
```

---

### 2순위: 월 광고비 500만원 이상, 멀티채널 집행 중

메타 + 구글 or 메타 + 네이버를 동시에 집행하는 업체는 채널별 기여도가 헷갈린다. 이 니즈가 스탠다드 계약으로 이어진다.

접근 방식: 채널 통합 트래킹 니즈를 건드린다.
```
"지금 두 채널 동시에 집행하시면 같은 고객한테 광고가 중복으로 가는 경우 많아요.
어느 채널이 실제 전환에 기여했는지 알 수 없는 상태인데,
이게 정리되면 예산 배분이 완전히 달라질 수 있어요."
```

---

### 3순위: D2C 브랜드로 성장 중인 인스타 계정

팔로워 1만~5만 구간에서 매출이 나오고 있고 광고를 늘리려는 시점. 이 타이밍에 데이터 인프라 이야기를 꺼내면 "그게 필요하다는 건 알았는데"라는 반응이 많이 나온다.

접근 방식: 성장 프레임으로 접근.
```
"지금 광고 늘리기 전에 측정 기반 먼저 만들어두는 게 나중에 비용을 훨씬 아껴요.
데이터 없이 광고 스케일업 하면 어디서 터지는지 몰라요."
```

---

### 4순위: 경쟁사(대형 대행사) 불만 있는 업체

"대행사가 리포트는 주는데 뭔 말인지 모르겠다", "비용 대비 뭐가 좋아졌는지 모르겠다" — 이런 말이 나오면 교체 니즈다.

접근 방식: 이해 가능한 리포트를 강조.
```
"저희 리포트는 '이 캠페인은 이번 달 매출에 기여했고, 이건 없애도 된다'는 식으로
실제 액션으로 이어지게 써드려요. 숫자 나열 말고요."
```

## 4. 리드 추적 템플릿

새 리드 발굴할 때마다 아래 테이블에 기록한다. 엑셀이나 노션에 복붙해서 쓰면 된다.

| 업체명 | 사이트 URL | 인스타 | GA4 세팅 | 픽셀 세팅 | CAPI 세팅 | 추정 월 광고비 | 추천 패키지 | 최초 연락일 | 연락 채널 | 진행 상태 | 담당자 | 메모 |
|--------|----------|--------|---------|---------|---------|-------------|-----------|-----------|---------|---------|------|------|
| OOO | example.com | @brand | X | O (이벤트 누락) | X | 500~1000만 | 스탠다드 | 2026-04-20 | DM | 첫 연락 완료 | 김OO | 광고 라이브러리 확인, 3개 캠페인 집행 중 |

### 진행 상태 분류

- `발굴`: 리드 확인만 된 상태
- `첫 연락 완료`: DM/이메일/전화 한 번 함
- `답변 대기`: 연락 후 회신 기다리는 중
- `미팅 잡힘`: 일정 확정
- `미팅 완료`: 상담 진행함
- `제안서 발송`: 패키지 제안서 보냄
- `계약 진행 중`: 계약서 검토 단계
- `계약 완료`: 클라이언트 전환
- `거절`: 이번에 안 됨 (3개월 후 재접촉 기록)
- `보류`: 나중에 다시 연락 요청받음

## 5. 아웃리치 첫 메시지 템플릿

### DM 첫 메시지 (인스타 / 카카오)

```
안녕하세요, [브랜드명] 대표님!
[브랜드명] [제품명] 보고 연락드렸어요.
광고 꾸준히 집행하고 계시던데, 혹시 지금 GA4랑 메타 CAPI 세팅은 
되어 있으신가요?

저희가 이커머스 브랜드 데이터 인프라 쪽 전문으로 하는데,
사이트 잠깐 봤더니 [발견한 문제 1가지]가 있더라고요.
5분만 이야기 나눠봐도 될까요?
```

**포인트:** 첫 문장에 브랜드명과 제품명 넣어서 개인화. 문제 1가지만 콕 집어서 구체성 확보. CTA는 부담 없는 "5분 이야기"로.

---

### 이메일 첫 메시지

```
제목: [브랜드명] 사이트 GA4 데이터 관련 (1분만 봐주세요)

안녕하세요, 저는 이커머스 데이터 분석 전문 LIANCP의 [이름]입니다.

[브랜드명] 사이트를 잠깐 살펴봤는데,
현재 [GA4 미세팅 / 픽셀 이벤트 누락 / CAPI 미연동] 상태로 보여서 연락드렸어요.

지금 광고 집행하고 계신 상황에서 이 부분이 방치되면,
메타 광고 전환 데이터의 최대 40%가 잡히지 않을 수 있어요.

혹시 이번 주 15분 정도 간단히 현황 공유해드려도 될까요?
비용이나 의무 없이 현재 세팅 상태만 한 번 봐드릴게요.

[이름] 드림
```

**포인트:** 제목에 브랜드명 + 구체적 행동 유도. 본문은 3단락 — 현황 진단, 리스크 제시, CTA. "비용/의무 없이"로 진입 장벽 제거.

## 부록: 카테고리별 타겟 찾기 빠른 경로

| 카테고리 | 리드 발굴 경로 |
|---------|-------------|
| 뷰티/스킨케어 | 화해 입점 브랜드 → 자체몰 보유 여부 확인 |
| 패션/의류 | 무신사 입점 브랜드 중 자체몰 보유 → 인스타 광고 집행 확인 |
| 건강식품 | 쿠팡 로켓그로스 상위 → 자체 브랜드몰 확인 |
| 반려동물 | 인스타 반려동물 계정 팔로워 1만+, 쇼핑 링크 보유 |
| 홈리빙 | 오늘의집 입점 브랜드 중 자체몰 보유 |
| 식품 | 마켓컬리 입점 브랜드 중 자체몰 + 메타 광고 집행 |
