> 재원 리서치 결과 2026-04-20

## 1. Store Leads
- **URL**: https://storeleads.app
- **특징**: 이커머스 쇼핑몰 1,380만개 DB. 뷰티/스킨케어 카테고리 필터 + 월매출 추정치 + 이메일 포함
- **가격**: $0.0039/리드부터
- **적용**: 한국 Shopify 뷰티몰 필터링 가능. 아임웹/Cafe24 커버 여부 확인 필요
- **우선순위**: 규모 커지면 1순위 유료 전환 대상

## 2. Apollo.io
- **URL**: https://www.apollo.io
- **특징**: 275M 컨택, 73M 기업. 이커머스 카테고리 필터 가능. 이메일+전화+회사정보
- **가격**: 무료 플랜 있음 / 유료 $49~99/월
- **적용**: 뷰티 브랜드 마케팅 담당자/대표 직접 타겟. 무료로 시작 가능
- **우선순위**: 지금 당장 무료로 테스트 가능
