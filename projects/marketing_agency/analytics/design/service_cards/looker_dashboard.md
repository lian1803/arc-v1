# Looker Studio 대시보드 구축

---

## 이게 왜 필요한가

GA4, 메타 광고, 구글 애즈, 네이버 광고 — 데이터가 플랫폼마다 따로 있다. 메타 들어가서 확인하고, 구글 들어가서 확인하고, GA4 들어가서 확인하고. 매주 이걸 엑셀로 합치는 데만 몇 시간이 간다.

Looker Studio는 이 데이터를 한 화면에 자동으로 모아주는 대시보드다. 아침에 링크 하나 열면 어제 성과, 이번 달 누적, 채널별 ROAS가 다 보인다.

실무에서 자주 겪는 문제:
- 광고 플랫폼마다 숫자가 다름 → 어느 게 맞는지 판단 불가
- 성과 보고서 만드는 데 매주 반나절 소모
- 팀원 / 대표님이 각자 다른 플랫폼 들어가서 다른 숫자 보고 다른 판단을 함
- 어떤 채널이 실제로 돈을 벌어오는지 한눈에 안 보임

---

## 우리가 해주는 것

- 데이터 소스 연결
  - GA4 (사이트 트래픽 + 전환)
  - Meta Ads (광고 비용 + 전환)
  - Google Ads (검색 광고 성과)
  - 네이버 광고 / 카카오 (스탠다드 이상)
- 대시보드 구성
  - 스타터: 기본 1개 (트래픽 + GA4 전환 요약)
  - 스탠다드: 2~3개 (채널별 성과 비교 + ROAS + 유입 경로)
  - 프리미엄: 5개 이상 (역할별 맞춤 — 대표용 / 마케터용 / 채널별)
- 자동 갱신 설정 (매일 자동으로 데이터 업데이트)
- 기간 필터, 채널 필터 조작법 교육 (1회)
- 대시보드 공유 링크 제공 (팀원, 대표님 공유 가능)

---

## 없으면 어떤 일이 벌어지나

- 성과 보고서 작성에 매주 2~4시간 소모 → 연간 100시간 이상
- 데이터가 분산돼 있으니 빠른 판단 불가 → 광고 최적화 타이밍을 놓침
- "이번 달 메타 ROAS 얼마야?" 라는 질문에 즉시 답하지 못하는 팀
- 대표님과 마케터가 서로 다른 숫자 기준으로 대화 → 방향이 안 잡힘

---

## 가격

| 패키지 | 대시보드 수 | 연결 채널 | 월 가격 |
|--------|------------|-----------|---------|
| 스타터 | 1개 (GA4 기본) | GA4 | 50~80만원 |
| 스탠다드 | 2~3개 (멀티채널) | GA4 + 메타 + 구글 + 네이버 + 카카오 | 120~180만원 |
| 프리미엄 | 5개 이상 (역할별 커스텀) | 전 채널 + BigQuery | 250~350만원 |

---

## 소요 기간

- 기본 대시보드 (GA4 단독): 3~5 영업일
- 멀티채널 대시보드: 7~10 영업일
- 커스텀 레이아웃 + 역할별 분리 구성: 14~20 영업일

---

## 주의사항

- 광고 플랫폼 계정 접근 권한 필요 (각 플랫폼 관리자 초대)
- Looker Studio는 구글 계정 필요 (기존 계정 사용 가능, 신규 생성도 무료)
- 데이터 연동은 실시간 아님 — GA4는 최대 24~48시간, 광고 플랫폼은 1일 단위 갱신
- 네이버 광고 데이터는 API 연동 방식 — 계정 설정 필요
- 한번 만들어진 대시보드 수정은 월 3회까지 포함 (추가 수정 건당 청구)
