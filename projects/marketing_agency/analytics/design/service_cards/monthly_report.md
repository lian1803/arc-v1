# 월 리포트 자동화

---

## 이게 왜 필요한가

광고는 매달 돌아가는데 "이번 달 잘 된 건지 아닌지"를 명확히 말할 수 있는 사람이 팀 안에 없다면 — 그 광고비는 사실상 검증 없이 나가는 거다.

매달 성과를 정리하는 월 리포트가 있어야:
- 어떤 캠페인이 효과 있었는지 기록이 남는다
- 다음 달 예산 배분을 데이터 기반으로 결정할 수 있다
- 외부 에이전시나 프리랜서에게 "이 달 성과가 이렇다, 왜?"를 물을 근거가 생긴다
- 대표님이 매달 "이번 달 어땠어?"라고 물을 때 숫자로 대답할 수 있다

문제는 리포트 만드는 게 노력이 크다는 것이다. 플랫폼마다 데이터 뽑고, 엑셀에 붙여넣고, 표 만들고, 코멘트 달고 — 매달 반나절 이상이 간다. 자동화하면 데이터 수집 + 기본 표는 자동으로 만들어지고, 인사이트만 추가하면 된다.

---

## 우리가 해주는 것

- 월 리포트 템플릿 설계 (브랜드 목표에 맞춘 KPI 기준 정의)
- 데이터 자동 수집 설정
  - GA4 → 트래픽, 전환, 유입 경로
  - 메타 / 구글 / 네이버 / 카카오 → 광고비, 전환, ROAS
  - 채널별 비교 자동 계산
- Looker Studio 기반 자동 리포트 생성 (매월 1일 자동 갱신)
- 리포트 형식
  - 스타터: 수동 작성 (기본 수치 정리 + 코멘트)
  - 스탠다드: 반자동 (데이터 자동 + 인사이트 수동 추가)
  - 프리미엄: 완전 자동화 (데이터 + 인사이트 + 다음 달 제언까지 자동 생성)
- 월 1회 리포트 공유 + 핵심 코멘트 전달 (이메일 또는 메신저)
- 스탠다드 이상: 월 1회 30분 리포트 리뷰 통화

---

## 없으면 어떤 일이 벌어지나

- 지난달 어떤 광고가 잘 됐는지 기억에만 의존
- 광고비가 매달 비슷하게 나가는데 개선 근거가 없음
- "이번 달 메타 ROAS 어때?" → "아... 좀 그랬어요" → 이게 의사결정 전부
- 6개월 후 "왜 이때 예산 줄였지?"라는 질문에 아무도 답을 못 함
- 에이전시에게 "성과가 없다"고 말하고 싶어도 데이터가 없어서 근거 제시 불가

## 가격

| 패키지 | 리포트 방식 | 포함 항목 | 월 가격 |
|--------|------------|-----------|---------|
| 스타터 | 수동 작성 | 기본 GA4 + 메타 수치 정리 | 50~80만원 |
| 스탠다드 | 반자동 (데이터 자동 + 코멘트 수동) | 멀티채널 + 인사이트 + 월 1회 리뷰 통화 | 120~180만원 |
| 프리미엄 | 완전 자동화 | 전 채널 + AI 인사이트 + 전략 제언 + 월 2회 컨설팅 | 250~350만원 |

---

## 소요 기간

- 리포트 템플릿 초기 설계: 5~7 영업일
- 첫 번째 리포트 발행: 템플릿 확정 후 해당 월 마감 시
- 자동화 설정 완료 후 다음 달부터 정기 발행

---

## 주의사항

- 첫 달은 데이터 수집 기간 → 두 번째 달부터 완전한 리포트 제공
- 리포트 기준 KPI는 계약 초기에 함께 정의 (ROAS, CPA, 전환수, 유입 트래픽 등)
- 플랫폼 계정 접근 권한 필요 (읽기 권한으로 충분)
- 리포트 형식 중간 변경 가능 (분기 1회 무료 수정)
- 완전 자동화 리포트는 AI 기반 인사이트 포함 — 최종 판단은 담당자 확인 필수
