### 주요 리스크

**1. GA4 ID 2개 동시 감지 — 이중 계측(Double Counting) 위험**
- 감지값: `G-07V7RKPJ1M` / `G-598429`
- `G-598429`는 ID 형식이 비정상적으로 짧음 (표준 GA4 ID는 `G-XXXXXXXXXX` 10자리). 구버전 UA 잔존 태그이거나 오기입 가능성 존재.
- 비즈니스 임팩트: 세션수·전환수 중복 집계 시 ROAS 수치 왜곡 → 광고 예산 배분 판단 오류 직결. 실제 전환이 1건인데 2건으로 보고될 수 있음.

**2. 전환 이벤트 1개(`conversion`)만 감지 — 퍼널 단계별 데이터 공백 가능성**
- D2C 뷰티 커머스 기준 최소 필수 이벤트: `view_item` / `add_to_cart` / `begin_checkout` / `purchase` / `view_item_list` — 이 중 정적 스캔에서 확인된 항목 0개.
- `conversion`이라는 이벤트명은 GA4 표준 이벤트명이 아님. GTM 커스텀 태그일 가능성 높으나, 실제 어떤 행동에 트리거되는지 확인 불가.
- 비즈니스 임팩트: 퍼널 이탈 지점 식별 불가 → 제품 상세 조회 후 이탈인지, 장바구니 이탈인지 구분 불가 → 개선 액션 방향 설정 불가.

**3. Meta Pixel 이벤트 매핑 검증 불가 — Meta 광고 최적화 신호 품질 미확인**
- Pixel ID `483166674335780` 설치는 확인됨.
- `Purchase`, `AddToCart`, `InitiateCheckout` 이벤트의 실제 발화 여부 및 value/currency 파라미터 포함 여부는 확인 불가.
- 비즈니스 임팩트: 팔로워 20,300명 + 화해 뷰티어워드 수상 브랜드로 Meta 광고 활용 가능성 높음. Pixel 이벤트 품질 저하 시 Meta 알고리즘 학습 신호 부족 → CPM·CPA 상승 직결.

**4. dataLayer 구조 내용 미검증 — 데이터 전달 품질 불명확**
- dataLayer 존재는 확인됨. 그러나 실제 push되는 변수(상품ID, 가격, 카테고리 등) 구성은 확인 불가.
- 비즈니스 임팩트: dataLayer가 비어있거나 불완전하면 GTM 태그가 존재해도 정확한 데이터 수집 불가. 수집된 것처럼 보이나 내용이 없는 '껍데기 트래킹' 상태일 수 있음.
