# 크몽 상품 5개 (paste-ready)

> 가격: 자동화 30/80/150, 웹사이트 200/500. 그대로 등록.
> 비유: 미용실 메뉴판 = 컷/펌/염색/코스 — 상품 5개 = 조합 5개.

## 상품 1. 자동화 베이직 (30만원)

**제목**: 엑셀·구글시트 자동화 — 단순 작업 1건

**설명**
반복되는 엑셀 작업, AI 도구로 1주 내 자동화합니다.

**포함**
- 자동화 스크립트 1개 (Python / Apps Script / VBA 중 선택)
- 사용법 매뉴얼 1장 (한국어)
- 7일 무료 수정

**적합한 의뢰**
- 매주/매일 반복하는 엑셀 정리 작업
- 여러 시트 합치기 / 데이터 정렬
- 보고서 자동 생성 (단순 형식)

**가격**: 30만원 (고정가)
**납기**: 5일

**미적합**
- 외부 API 연동 필요 시 → 상품 2 (자동화 스탠다드)
- DB 연결 / 멀티 사용자 → 상품 3 (자동화 프리미엄)

**포함**

- 사용법 매뉴얼 + 14일 무료 수정

**적합한 의뢰**
- 네이버 블로그 자동 포스팅
- 카톡 자동 답변

**가격**: 80만원
**납기**: 10일

## 등록 방법 (Lian)

1. 크몽 로그인 → 전문가 페이지 → "서비스 등록"
2. 카테고리 = "IT·프로그래밍 → 업무 자동화" (상품 1-3) / "웹사이트 개발" (상품 4-5)
3. 제목/설명/가격/납기 그대로 paste
4. 썸네일 1장 (Claude Code 로 생성 또는 무료 stock)
