# 위시켓 입찰글 템플릿 2개 (paste-ready)

> 사용법: 변수 [대괄호] 채우기 (2분/건). 첫 글 quality 80% 가능.
> 비유: 같은 자기소개서 회사 이름만 바꾸기.

## 템플릿 A. 자동화 카테고리

```
[CLIENT_NAME] 님 안녕하세요.

[PROJECT_TITLE] 의뢰글 잘 봤습니다. [USER_PAIN_KEYWORD] 문제 해결, 제가 가능합니다.

**진행 가능 이유**
- 비슷한 자동화 프로젝트 [N]건 진행 (포트폴리오: [LINK])
- 평균 납기: [DAYS]일
- 단가: [PRICE]만원 (고정가, buffer 30% 박혀 있어 추가 비용 없음)

**제안 단계**
1. 30분 디스커버리 콜 → 의뢰 내용 정확히 파악
2. 24시간 내 상세 견적서 + 일정 송부
3. 50% 선금 후 착수 / 납기 시 50%
4. 납기 후 7일 무료 수정

[CALENDLY_LINK]

위시켓 메시지로 답장 주세요. 24시간 내 응답합니다.

감사합니다.
```

**변수 가이드 (자동화)**
- [CLIENT_NAME]: 의뢰글 상단 의뢰사 이름
- [PROJECT_TITLE]: 의뢰글 제목 그대로
- [USER_PAIN_KEYWORD]: 의뢰글에서 가장 많이 나오는 명사 ("엑셀", "보고서", "수동")
- [N]: 포트폴리오 개수 (월 1주차 = 3, 이후 ramp)
- [LINK]: 포트폴리오 GitHub/Notion URL
- [DAYS]: 자동화 = 5-10, 단순 = 3-5
- [PRICE]: 의뢰 예산의 70-80%
- [CALENDLY_LINK]: 미팅 link

## 템플릿 B. 웹사이트 카테고리

```
[CLIENT_NAME] 님 안녕하세요.

[PROJECT_TITLE] 의뢰글 봤습니다. [PAGES_KEYWORD] 페이지 + [FEATURE_KEYWORD] 기능, 제가 가능합니다.

**진행 가능 이유**
- 비슷한 사이즈 웹사이트 [N]건 진행 (포트폴리오: [LINK])
- 평균 납기: [DAYS]일 (위시켓 평균 4주 대비 단축)
- 단가: [PRICE]만원 (고정가, 추가 비용 없음)

**포함 사항**
- 반응형 (PC/모바일/태블릿)
- 기본 SEO (meta + sitemap)
- 결제 연동 (필요 시 토스/포트원)
- 도메인 연결 + SSL
- 운영 매뉴얼 1장 (Lian 직접 수정 가능하게)

**미포함 (별도 견적)**
- 디자인 시안 별도 (시안 보유 시 무료, 미보유 시 +30만)
- 콘텐츠 작성 (텍스트/이미지)
- 1개월 후 유지보수

**제안 단계**
1. 30분 디스커버리 콜 → 페이지 구조 파악
2. 48시간 내 와이어프레임 + 견적서 송부
3. 50% 선금 후 착수
4. 중간 미리보기 1회 + 납기 시 50%

[CALENDLY_LINK]

24시간 내 답장합니다.

감사합니다.
```

**변수 가이드 (웹사이트)**
- [PAGES_KEYWORD]: 의뢰글 페이지 수 ("5페이지", "10페이지")
- [FEATURE_KEYWORD]: 핵심 기능 ("결제", "회원가입", "예약폼")
- [N]: 포트폴리오 개수
- [DAYS]: 웹사이트 = 14-21
- [PRICE]: 의뢰 예산의 75-85%

---

## 입찰 cadence

- 월/화: 5건씩 입찰 (총 10)
- 수: 답장 모니터 + 추가 5건 (총 15)
- 목/금: 답장 우선, 추가 입찰 0-3건

## 답장률 가이드 (Hypothesis)

- 첫 주: 5-7%
- 4주차: 10-12% (포트폴리오 채워지면)
- 8주차: 15%+ (후기 쌓이면)

5% 미만 = 템플릿 변경 (포지셔닝 결정 1 다른 옵션)
