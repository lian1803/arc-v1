## 4개 파일 요약

### 01_business_reality.md
- 한국 디지털 마케팅 시장: USD 7.8B (2024) → 19.7B (2033)
- 자영업자 마케팅 대행 지속가능 단가: 월 200~300만원
- 폐업자: 2024년 100만명, 5년 생존율 40.2%
- 미용실 생존율 91.6% (식당 80.6% 대비 +11%)
- 실제 자영업자 도구: 네이버 플레이스 (ROI 3.2배) > 인스타 > 블로그 > 당근 > 카카오

### 02_b2b_sales.md
- 콜드 이메일 응답률: 5% → 고급화 18%, 3회 팔로우업 22%
- SPIN Selling (자영업자 특화): 자영업자가 스스로 문제 깨닫게 함
- 최적 길이: 75~125단어
- 자영업자 특화: 예산 대신 "월 손님 수", 마케팅 어려움 대신 "신규 손님 출처"

### 03_psychology.md
- 손실회피: 이득의 3배 강함 (Kahneman)
- Self-Image Type A/B: 메시지 톤 180도 다름
- 초이스 오버로드: 선택지 3개 이상 = 결정 불능
- 현재편향: 미래보다 현금흐름 우선

### 04_self_employed_voice.md
- 네이버 카페 "아프니까 사장이다" (60만) + YouTube + Instagram 분석
- 자영업자 최우선: "이번 달 신규 손님" (현재편향)
- 신뢰도: 과거 대행사 경험 80% 결정
- 원하는 메시지: "뭘 해야 하는지 정확히" (선택지 제약)

## 12개 핵심 Takeaway

### 영역 1 (사업 현실)

**1. 월 200~300만원이 유일한 지속가능 단가**
- 자영업자 심리적 임계값: "월급 수준의 투자"
- 메시지 적용: 가격 제시 시 "월 200만원" 중심

**2. 자영업자 손에 닿는 채널 4개 (웹UI 중심, 모바일 우선)**
- 터미널, 개발자도구, CLI 불가능
- 메시지 적용: "터미널 안 열어도 돼, 폰과 네이버만 있으면"

**3. 미용실 특화 우선 (91.6% 생존율, 정기 예약)**
- 식당은 빠른 결과 필요 (신규손님 매달 필수)
- 메시지 적용: 미용실 → "월 신규 예약 +15명", 식당 → "월 신규 손님 +30명"

### 영역 2 (영업)

**4. SPIN Selling 진단 = 자영업자 스스로 문제 깨닫게 함**
- "당신이 월 얼마를 손실하고 있는지 알아요?"
- 메시지 적용: 콜드 메일 무조건 SPIN S/P 질문으로 시작

**5. 콜드 이메일: 5% → 3회 팔로우업으로 22%**
- 최적 길이 100단어 이내
- 메시지 적용: 팔로우업 시퀀스 (1차: 문제 → 2차: 손실 → 3차: 사회적 증거)

**6. SPIN을 자영업자 일상 언어로 리프레이즈**
- 예산 X, "월 손님 수" O
- 메시지 적용: 질문을 원, 손님 수, 월 손실액으로 표현

### 영역 3 (심리)

**7. 손실회피 = 이득의 3배 강함**
- "월 -100만원 손실" > "월 +30만원 수익"
- 메시지 적용: 첫 줄은 손실액 (구체적)

**8. 자존심 Type A/B = 메시지 톤 180도 다름**
- Type A (나는 안다): "콘텐츠는 좋은데 알고리즘만" (자존심 보호)
- Type B (나는 모른다): "당신은 아무것도 안 해도" (간편성)
- 메시지 적용: Type 판별 후 톤 전환

**9. 초이스 오버로드 = 선택지 3개 이하로 제약**
- 5개 채널 제안 = 선택 못 함
- 3개 채널 제안 = 즉시 수락
- 메시지 적용: 선택지 3개 이상 금지, 정답 형태로 제시

### 영역 4 (Voice)

**10. 최우선은 "이번 달 신규 손님" (현재편향)**
- 미래 전략 X, 현금흐름 긴급
- 메시지 적용: "3개월 후" X, "첫 1주일 효과" 강조

**11. 신뢰도 80% = 과거 대행사 경험**
- 신뢰 형성 = 투명성 + 빠른 응답
- 메시지 적용: "주 1회 카톡 리포트" + "24시간 응답" 약속

**12. "뭘 해야 하는지 정확히" (선택지 제약)**
- 자영업자는 선택권 X, 정답 원함
- 메시지 적용: 모호한 제안 제거, "매주 월요일 10시 리포트" 구체성

## 메시지 생성 기능 개선 Map (Takeaway → Code)

|---|---|---|
| 1-3 (사업 현실) | 범용 톤 | 자영업자 특화 (vertical 분기) |

| 7-9 (심리) | 이득 중심 | 손실 우선 + Type A/B 분기 |
