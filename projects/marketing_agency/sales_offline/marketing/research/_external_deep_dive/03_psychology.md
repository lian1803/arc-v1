# 영역 3: 심리 (구매 결정 + 영업 메시지 응답)

**결론**: 자영업자 구매 심리는 손실회피 (Kahneman) > 이득 추구, 초이스 오버로드 (IDEO) + 현재편향 (present bias) + 자존심 (self-image)의 4가지 축으로 움직인다. 메시지는 자영업자의 자존심 상태 (내가 마케팅 전문가 vs 나는 몰라)에 따라 **톤을 완전히 바꿔야** 효과적이다.

---

## 1. Robert Cialdini 6 영향 원칙 (자영업자 버전)

### 1.1 Reciprocity (호혜성)
**원칙**: 사람은 받은 것을 돌려주려는 심리

**자영업자 적용**:
```
❌ "우리 서비스 좋으니까 써보세요"
✅ "우선 무료로 당신 매장 네이버 플레이스 점수 진단해드릴게요"
```
- 무료 진단 제공 → 자영업자가 "고마운 마음"으로 상담 수락
- 호혜의 심리: "공짜로 주는 거 받았으면 뭔가 해줘야지" (의무감)

### 1.2 Commitment & Consistency (약속과 일관성)
**원칙**: 한 번 약속하면 그 약속을 지키려는 심리

**자영업자 적용**:
```
❌ "계약하면 월 200만원입니다"
✅ "이번 주 3일 무료 진단받고 나서 계약할지 결정하세요"
   → (3일 후) "당신이 한 약속: '효과 있으면 시작한다'"
```
- 작은 약속 (무료 진단 수락) → 큰 약속 (계약)으로 확대

### 1.3 Authority (권위)
**원칙**: 전문가/권위 있는 사람의 말을 믿음

**자영업자 적용**:
```
❌ "우리가 최고입니다"
✅ "같은 동네 미용실 8곳이 우리 손님인데... (사진 보여주기)"
```
- 권위 = 업계 최고 수상이 아니라 "내 동네에서 검증된 사람"
- 자영업자 입장: "저 미용실도 이 사람 믿는데, 내가 안 믿을 이유가?" 

### 1.4 Social Proof (사회적 증거)
**원칙**: 다른 사람들이 하는 행동을 따라함

**자영업자 적용**:
```
❌ "고객 만족도 98%"
✅ "최근 3개월간 네이버 플레이스 리뷰 +30개 달성한 식당이 3곳"
```
- 리뷰 개수, 별점, 예약 증가 같은 "구체 수치" 필수
- "내 동네에서 실제 일어난 일" > "업체의 주장"

### 1.5 Liking (호감)
**원칙**: 자신과 비슷하고 호감 가는 사람의 제안을 받아들임

**자영업자 적용**:
```
❌ "저는 마케팅 전문가입니다"
✅ "저도 5년 전 작은 식당 했어서, 사장님 상황 정확히 압니다"
```
- 자영업자는 "마케팅 전문가"보다 "자영업 경험자"를 더 신뢰
- 공통점 강조: 같은 고통, 같은 시간 부족, 같은 돈의 가치 관점

### 1.6 Scarcity (희소성)
**원칙**: 드문 것, 빠져나갈 위험이 있는 것에 더 가치를 둠

**자영업자 적용**:
```
❌ "언제든 시작 가능해요"
✅ "이번 달 5개 매장만 받고 있는데, 자리 남았으면 시작할 수 있어요"
```
- 시간 제한: "이번 주 신청하면 초기 할인 30%"
- 개수 제한: "동네당 3곳까지만 받습니다" (경합사 포화 방지 + 희소성)

**주의**: 자영업자는 "가짜 희소성" 잘 봄 ("매일 시간 제한이면 제한이 아님")

## 2. Daniel Kahneman 손실회피 (Loss Aversion)

### 2.1 프로스펙트 이론 (Prospect Theory)
**핵심**: 이득 100만원보다 손실 100만원을 **3배** 더 크게 느낌

**자영업자 적용**:

| 메시지 | 반응도 |
|--------|--------|
| "월 매출 +30만원 증가" | 낮음 (무심함) |
| "월 신규 손님 +10명" | 중간 |
| **"매장이 검색 안 뜨면 월 500만원 손실이잖아요"** | **매우 높음** |

- 자영업자는 "먹고사는 것"이 우선
- 이득보다 "손실 회피"가 심리적 강도 3배

### 2.2 손실회피를 영업에 적용

**패턴 1: 현상 유지 편향 (Status Quo Bias)**
```
❌ "새로운 마케팅 전략을 도입하세요"
✅ "지금 네이버 검색 안 뜨는 상태를 유지하면 월 500만원 손실되는데,
    이걸 0으로 만들 수 있어요"
```

**패턴 2: 구체적 손실액 명시**
```
❌ "마케팅이 중요해요"
✅ "사장님, 지금 당신 가게 검색 안 뜨니까
    - 신규 손님: 0명
    - 월 손실: 500만원 (평균 고객단가 × 예상 손실 손님수)
    이 맞나요?"
```

## 3. IDEO HCD (Human Centered Design) - Choice Overload 해결

### 3.1 초이스 오버로드 (Choice Paralysis)
**현상**: 선택지가 많으면 결정을 못 함

**자영업자 입장**:
```
"네이버 플레이스도 해야 하고, 인스타도 해야 하고, 
 블로그도 해야 하고, 당근마켓도...?"
→ "다 너무 많아. 아무것도 안 하겠다"
```

### 3.2 제약 프레이밍 (Constraint Framing)
**원칙**: "다 하지 말고, 이것만 하세요" → 결정 쉬움

**자영업자 적용**:
```
❌ "종합 마케팅 패키지: 네이버, 인스타, 블로그, 당근, 카카오"
✅ "미용실 특화: 당근마켓 + 카카오맵만 관리해드릴게요.
    이것만으로 월 신규 예약 +15명 달성 가능"
```

- 자영업자는 선택지 줄 때 더 빠르게 결정
- "저희는 전문가니까, 당신은 최고의 채널 3개만 집중하세요"

## 4. 현재편향 (Present Bias) vs 미래이득

### 4.1 자영업자의 시간 선호도
- **오늘의 손실**: 매우 크게 느낌 (현금 흐름 긴급)
- **내일의 이득**: 거의 무시 (희망사항)

**자영업자 입장**:
```
"월 200만원 투자 → 3개월 후 효과" ?
아니라, "이번 주 투자 → 이번 주 효과"를 원함
```

### 4.2 현재편향 극복 전략

**전략 1: 초기 빠른 성과**
```
약속: "첫 2주 안에 당신 네이버 플레이스 점수 +10점"
     (실제로 가능한 quick win)
→ 자영업자: "오, 이 사람은 진짜 빠르네" → 신뢰 형성
```

**전략 2: 주간 리포트 (현실감 강화)**
```
"월 1회 보고"가 아닌 "주 1회 카톡 리포트"
- 월요일: "이번 주 목표"
- 금요일: "이번 주 성과" (리뷰 +2개, 방문자 +30명 등)
→ "아, 계속 뭔가 움직이고 있구나" 체감 중요
```

## 5. Seller Psychology: 자영업자의 Self-Image

### 5.1 2가지 Self-Image 타입

**Type A: "나는 마케팅을 잘 안다"**
- 이미 블로그/인스타 운영 중
- "내가 이미 충분히 잘 하는데 왜?"
- 메시지 전략: **자존심 보호** + 협력 프레임

**Type B: "나는 마케팅을 몰라"**
- 마케팅 전혀 안 함, 뭘 해야 할지 모름
- "전문가 말 들어야지"
- 메시지 전략: **간편성** + 초이스 축소

### 5.2 Type A 대상 메시지 (자존심 보호)

```
❌ "당신의 인스타는 반응이 낮아요"
✅ "당신의 인스타 콘텐츠 스타일은 정말 좋은데, 
    알고리즘만 최적화하면 +100% 도달율 가능해요"
```

- Type A는 "당신의 노력이 틀렸다" 느끼면 거절
- 대신 "당신은 잘 했는데, 이건 기술 영역" (분리)

### 5.3 Type B 대상 메시지 (간편성 강조)

```
❌ "마케팅 전략이 필요해요"
✅ "당신은 매장 운영만 하세요, 나머지 다 우리가 해요.
    월 2회 리포트만 받으면 끝"
```

- Type B는 "나도 뭐 해야 하나?" 질문 금지
- 대신 "당신은 아무것도 안 해도 돼" (완전 위임)

### 5.4 Self-Image Reset (심리 전환)

**위험한 메시지 (자존심 상함)**:
```
"당신은 마케팅을 모릅니다"
```

**효과적인 메시지 (자존심 유지)**:
```
"마케팅은 원래 자영업자 혼자는 불가능해요. 시간도 없고.
우리는 이 부분만 전담하고, 당신은 핵심 사업에만 집중하세요"
```

## 6. Cialdini + Kahneman 통합: 자영업자 메시지 프레임

### 6.1 메시지 구조 (황금 공식)

```
[호감] "저도 작은 카페 했었어요. 사장님 상황 알아요"
      ↓
[권위] "근데 혼자는 불가능하더라고요. 
        전문가랑 팀 짜니까 한 달에 효과 났어요"
      ↓
[손실회피] "요즘 마케팅 안 하면 경합사에 밀려요.
          당신 동네에도 5곳이 네이버 광고 돌리고 있죠?"
      ↓
[희소성] "우리 팀도 매달 5곳만 받아요. 자리 남았으면..."
      ↓
[사회적증거] "최근 미용실 2곳 계약했는데, 
             첫 달에 예약 +10명씩 달성했어요"
```

### 6.2 자존심 보호 + 손실회피 통합

```
"당신 인스타는 정말 잘 만들었는데,
 알고리즘 최적화 안 되면 월 100명 도달률 손실이잖아요?
 우리는 그 부분만 담당할게요, 당신은 콘텐츠만"
```

## 7. Behavioral Economics: 자영업자 특화 심리 원칙

### 7.1 Endowment Effect (보유 효과)
- 자영업자는 **자신의 가게 = 자신의 자산**으로 과평가
- "내 가게는 다른 곳과 달라" (사실상 별로 다르지 않음)
- **대책**: "당신 가게의 장점은... (구체)" → 자존심 만족

### 7.2 Sunk Cost Fallacy (매몰비용 오류)
- "이전에 광고비 500만원 썼는데 효과 없었어"
- "그래서 이건 다르다"는 설명해야 함
- **대책**: "이전엔 왜 안 됐나? (원인 분석) → 우리는 이걸 다르게" (신뢰 복구)

### 7.3 Status Quo Bias + Regret Aversion
- 자영업자는 변화를 피함 ("지금 이대로가 최선")
- 동시에 "내가 바꿨는데 실패하면?" 두려움
- **대책**: "3개월 보증제, 효과 없으면 환불" (리스크 제거)

## 핵심 Takeaway 3개

**1. 손실회피 = 이득의 3배 강함 (Kahneman)**
- 모든 메시지 첫 줄: 손실액 (월 X원 손실)
- 두 번째 줄: 이득 (월 Y원 개선)
- 우선순위: 손실 > 이득
- **메시지 적용**: "마케팅 안 하면 월 손실액은..." (구체 숫자 필수)

**2. 자존심 2가지 타입 (Type A: 나는 알아 vs Type B: 나는 몰라) → 톤 완전히 다름**
- Type A: "당신의 노력은 좋은데, 알고리즘만 우리가" (자존심 보호)
- Type B: "당신은 아무것도 안 해도 돼, 우리가 다" (간편성)
- 맞는 톤 못 찾으면 즉시 거절
- **메시지 적용**: 초기 상담에서 Type 판별 후 메시지 tone 180도 전환

**3. 초이스 오버로드 극복 = 선택지 3개 이하로 제약**
- "종합 패키지" 제안 X
- "당신 업종 특화 3채널" 제안 O (당근 + 카카오맵 + 리뷰 관리)
- 선택지 줄 때 결정 속도 5배 빨라짐
- **메시지 적용**: "미용실 사장님들은 이 3개만" (단순화 + 신뢰)
