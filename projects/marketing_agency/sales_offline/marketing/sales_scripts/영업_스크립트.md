# 1부. 현장 방문 스크립트

---

## 버전 1: 미용실 / 네일샵 / 피부샵 (예약 기반 업종)

> 접근 시점: 손님 없는 오전 10~11시, 또는 오후 2~4시
> 준비물: 진단 PDF (태블릿 or 출력), 원페이저 1장

---

### [입장 — 첫 5초]

**영업**: "안녕하세요, 죄송한데 사장님 계세요? 잠깐 말씀 드릴 게 있어서요."

*(사장님이 나오면)*

**영업**: "안녕하세요, 리안 컴퍼니에서 왔어요. 소상공인 마케팅 도와드리는 곳인데요. 요즘 네이버 플레이스 관련해서 드리고 싶은 자료가 있어서 잠깐 2~3분만 괜찮으실까요? 저 뭐 사달라는 거 아니에요, 먼저 진단 결과 보여드리고 싶어서요."

---

### [S — 상황 파악]

**영업**: "혹시 요즘 손님들이 어떻게 오세요? 단골분들도 계시겠지만, 새 손님들은 주로 어디서 찾아오시는지 혹시 아세요?"

**사장님**: "네이버에서 많이 오죠, 요즘은 거의 다 거기서 오는 것 같아요."

**영업**: "맞아요, 맞아요. 저도 그렇게 생각했어요. 혹시 네이버 플레이스 관리는 지금 따로 하고 계세요? 사진이라든지, 소식 이런 거요."

**사장님**: "아, 그냥 처음에 등록만 해놨고 거기까지는 잘... 바빠서요."

**영업**: "아, 맞아요. 대부분 그러세요. 그러면 지금 리뷰는 몇 개 정도 되세요?"

*(화면 보거나 사장님이 대략 답하면)*

---

### [P — 문제 인식]

**영업**: "아, 제가 오기 전에 잠깐 이 근처 미용실들 한번 봤거든요. 화면 잠깐 봐도 되세요?"

*(태블릿 or 휴대폰 꺼내서 진단 PDF 보여주기)*

**영업**: "지금 보시면요, 이쪽 양주 어둔동 기준으로 '미용실' 검색했을 때 상위에 뜨는 곳들이에요. 1위 업체가 리뷰가 지금 248개고요. 2위는 리뷰 187개예요. 사장님 업체는 지금 여기 보시면... 34개거든요."

**사장님**: "아, 그렇구나..."

**영업**: "그리고 리뷰 응대율 보시면, 1위 업체는 리뷰 달리면 거의 다 답장을 달아요, 92%. 사장님 업체는 지금 응대 기록이 없어서 0%로 잡혀있어요. 이게 순위에 영향을 줘요, 네이버가."

---

### [I — 함의, 숫자로 보여주기]

**영업**: "이 순위 차이가 실제로 어떤 의미냐 하면요. 지금 1위 업체 기준으로 월 플레이스 유입이 한 달에 한 800~1,000클릭 정도 돼요. 사장님 업체는 지금 여기 보면 유입이 거의 한 130~150 수준이에요. 거의 6~7배 차이나요."

**사장님**: "그게 그렇게 많이 차이나요?"

**영업**: "네. 그리고 이게 클릭이 아니라 실제 예약으로 환산하면, 1위 업체는 신규 손님이 한 달에 한 20~30명 플레이스에서 오는 거고요, 지금 순위대로면 사장님은 4~5명 수준이에요. 한 달에 15~20명 차이나요, 신규 손님이."

**사장님**: "..."

**영업**: "근데 이게 사장님 잘못이 아니에요. 그냥 리뷰 관리, 소식 등록, 사진 정비 이런 게 실제로 순위에 영향을 주는데, 이걸 신경 써줄 사람이 없으셨던 거거든요."

---

### [N — 필요, 해결책 제시]

**영업**: "저희가 이걸 관리해드리는 건데요. 지금 진단서 여기 보시면요, 개선하면 되는 항목들이 딱 나와요. 사진 보강, 소식 등록, 리뷰 응대, 이 세 가지만 잡아도 3개월 안에 순위 한 10~15계단 올라간 케이스들이 있어요."

**영업**: "저희 가장 기본 패키지가 월 38만원이에요. 한 달에 38만원으로 사장님 대신 플레이스 관리 전부 해드리는 거고요. 매달 리포트로 순위 어떻게 바뀌었는지, 리뷰 몇 개 붙었는지 보내드려요."

*(원페이저 건네주며)*

**영업**: "자세한 건 이거 보시면 되고요. 오늘 결정 안 하셔도 돼요. 진단서는 그냥 드리는 거예요. 한번 보시고, 궁금하신 게 생기면 카톡으로 연락 주세요."

*(명함 or 카카오 QR 건네기)*

---

### [사장님이 관심 보일 때 — 클로징 연결]

**사장님**: "얼마라고 하셨어요?"

**영업**: "가장 기본이 월 38만원이고요, 리뷰도 더 빨리 쌓고 싶으시면 56만원짜리가 있어요. 이건 블로그 체험단이랑 리워드 트래픽까지 포함돼요. 미용실이면 56만원짜리가 보통 더 효과 잘 나와요, 리뷰 수가 빨리 올라가니까요."

**영업**: "이번 달에 시작하시면 다음 달 리포트까지 한 번 보고 판단하실 수 있어요. 화요일이나 목요일 중에 계약서 잠깐 갖고 올까요, 30분이면 돼요."

## 버전 2: 음식점 / 카페 (유동인구 기반)

> 접근 시점: 점심 전 10:30~11:30, 또는 브레이크 타임 14:30~16:30
> 준비물: 진단 PDF, 경쟁 음식점 리뷰 비교 화면 캡처

---

### [입장]

**영업**: "안녕하세요, 바쁘실 텐데 죄송해요. 잠깐 2분만 시간 주실 수 있을까요? 이 근처 맛집 네이버 현황 관련해서 드릴 자료가 있어서요."

*(사장님이 잠깐 여유 있어 보이면)*

**영업**: "저 리안 컴퍼니인데요, 소상공인 플레이스 관리 해드리는 곳이에요. 오기 전에 이 근처 '양주 맛집' '양주 카페' 검색해봤거든요. 요즘 배달 말고 홀 손님 더 늘리고 싶으신 분들한테 도움이 많이 됐어요."

---

### [S — 상황 파악]

**영업**: "요즘 홀 손님이 어디서 많이 오세요? 당근이나 배달의민족 말고, 직접 검색해서 오시는 분들이요."

**사장님**: "아무래도 네이버죠. 친구들한테 소개받아서 오는 것도 있고요."

**영업**: "맞아요. 그 '소개받아서'가 사실 네이버 리뷰 보고 오는 경우가 엄청 많아요. '야 거기 리뷰 많고 별점 좋던데' 이렇게요. 혹시 지금 리뷰가 몇 개 정도 있으세요?"

---

### [P — 문제 인식]

**영업**: "제가 여기 근처 '양주 삼겹살' (또는 해당 업종) 한번 보여드려도 될까요? 지금 상위에 있는 업체들이에요."

*(화면 보여주며)*

**영업**: "1위 업체 리뷰 지금 342개고요, 사장님 업체가 41개예요. 리뷰 수 차이가 8배예요. 그리고 플레이스 소식, 저기는 이번 달에만 소식을 5개 올렸어요. 세트 메뉴 교체 공지, 주말 이벤트, 이런 거요. 사장님 업체는 마지막 소식이 5개월 전이에요."

**사장님**: "아, 그게 그렇게 중요한가요?"

**영업**: "네이버가 최근에 업데이트를 자주 한 플레이스를 위에 올려줘요. 그래서 소식을 꾸준히 올리는 게 순위에 영향을 줘요. 직접 하기 너무 번거로우신 거 아닌가 싶어서요."

---

### [I — 함의]

**영업**: "지금 검색 순위로 보면, 1위 업체는 네이버에서 한 달에 홀 신규 손님을 한 30~40명 정도 끌어오는 거고요. 사장님 위치에서는 한 7~8명 수준이에요. 한 달에 20명 차이면, 평균 객단가 25,000원 잡아도 월 50만원 차이나요."

**사장님**: "그게 그렇게 차이나요?"

**영업**: "배달이야 플랫폼 수수료 내시잖아요. 홀 손님은 마진이 더 높죠. 그 홀 손님을 네이버에서 가져오는 게 지금 순위 싸움이에요."

---

### [N — 해결책]

**영업**: "이걸 저희가 관리해드리면요. 소식 등록 월 3건, 리뷰 응대, 사진 정비, 이게 기본이에요. 월 38만원이고요. 만약 리뷰 빠르게 쌓고 싶으시면 블로그 체험단 포함된 56만원 패키지가 있어요."

**영업**: "음식점이나 카페는 보통 A(38만원)부터 시작해서 3개월 후에 B(56만원)로 올라가는 분들이 많아요. 먼저 검색이 되게 만들고, 그다음에 리뷰 쌓는 거예요."

**영업**: "진단서는 이거 그냥 가져가세요. 보시고 카톡 주시면 자세히 설명해드릴게요."

---

### [재방문 시나리오 — 관심 있었던 업체]

**영업**: "안녕하세요, 지난번에 진단서 드렸던 리안 컴퍼니예요. 혹시 보셨어요?"

**사장님**: "아, 네, 봤어요."

**영업**: "지난 3주 동안 보니까 사장님 근처 경쟁 업체 리뷰가 또 7개 늘었거든요. 사장님 리뷰는 지난번이랑 똑같이 41개예요. 격차가 계속 벌어지고 있어서, 이번 달 안에 시작하시는 게 좋을 것 같아서요."

## 버전 3: 학원 / 전문직 (신뢰도 기반)

> 접근 시점: 오전 수업 전 9~10시, 또는 오후 수업 전 14~15시
> 준비물: 키워드 순위 비교 화면, 블로그 노출 분석

---

### [입장]

**영업**: "안녕하세요, 원장님 계세요? 저 리안 컴퍼니에서 왔는데요, 소상공인 온라인 마케팅 도와드리는 곳이에요. 학원 관련해서 드릴 자료가 있어서 잠깐만 시간 주실 수 있을까요?"

---

### [S — 상황 파악]

**영업**: "요즘 등록 문의가 주로 어디서 와요? 지인 소개가 제일 많은가요, 아니면 온라인에서도 오는 편인가요?"

**사장님**: "대부분 소개예요. 온라인에서도 오긴 하는데 많진 않아요."

**영업**: "혹시 블로그나 네이버 플레이스는 지금 운영 중이세요?"

**사장님**: "플레이스는 등록은 돼 있고, 블로그는 예전에 조금 했는데 지금은 안 하고 있어요."

---

### [P — 문제 인식]

**영업**: "제가 '양주 영어학원' (또는 해당 업종) 검색해봤거든요. 지금 상위에 뜨는 학원들 보시면요, 1위가 블로그 포스팅이 100개 이상 있어요. 2위도 70개 넘고요. 그리고 플레이스 리뷰가 80~120개 사이예요. 학부모들이 결정할 때 리뷰 진짜 많이 봐요."

**사장님**: "그렇군요."

**영업**: "그리고 이게 진단서에도 나와있는데요, 원장님 학원 키워드가 지금 '양주 영어학원' 검색하면 몇 위에 뜨냐 하면, 플레이스 기준으로 14위예요. 1페이지가 보통 5~7개거든요. 지금 사람들 눈에 잘 안 보이는 위치예요."

---

### [I — 함의]

**영업**: "학부모들이 학원 찾을 때 1페이지 3~4개 보고 전화해요. 14위면 거의 전화를 안 받으시는 거예요. 지금 한 달에 플레이스에서 오는 문의가 몇 건이에요?"

**사장님**: "2~3건 정도인 것 같아요."

**영업**: "1위 업체는 한 달에 플레이스 문의만 12~15건이에요. 10건 차이나면 학생 3~4명 차이고, 수강료 평균 15만원이면 한 달에 45~60만원이에요."

---

### [N — 해결책]

**영업**: "학원은 신뢰도가 핵심이에요. 리뷰 수보다는 리뷰 내용이 좋아야 하고, 플레이스 사진이랑 소개글이 전문적으로 보여야 해요. 저희가 그 부분을 잡아드리고요."

**영업**: "블로그 체험단은 학원의 경우 좀 신중하게 해야 해서, 저희는 플레이스 최적화 + 리뷰 관리 + 소식 등록 기본부터 추천드려요. 월 38만원이에요. 3개월 하면 순위 변화 직접 보실 수 있어요."

**영업**: "진단서 가져가세요. 거기 지금 순위 기록돼 있으니까, 3개월 후에 비교해보시면 차이 직접 아실 거예요."

# 2부. 카카오톡 DM 시퀀스

---

## 버전 A: 진단 결과 전달형 (콜드 — 첫 접촉)

> 사전 조건: 진단 PDF 생성 완료, 업체 카카오 채널 or 사장님 번호 확보

---

### 1차 DM (첫 접촉)

```
안녕하세요, 리안 컴퍼니예요.
[업체명] 네이버 플레이스 무료 진단해봤는데요,
경쟁사 대비 개선 포인트 3가지 찾았어요.
PDF로 정리했는데 보내드려도 될까요? 🙏
```

*(답장 오면 PDF 파일 전송 + 아래 한 줄 추가)*

```
PDF 보내드렸어요.
오른쪽 위 빨간 표시 부분이 지금 당장 올릴 수 있는 순위예요.
궁금하신 거 있으면 편하게 여쭤보세요!
```

---

### 2차 DM (2일 후 — 반응 없을 때)

```
안녕하세요, 이틀 전에 진단서 보내드린 리안 컴퍼니예요.
```

# 부록. 현장 대화 흐름 요약 카드

> 출력해서 가방에 넣어두는 용도

```
┌─────────────────────────────────────────┐
│  SPIN 현장 흐름                          │
├─────────────────────────────────────────┤
│                                         │
│  S — "요즘 손님 어디서 오세요?"           │
│      → 네이버/플레이스 얘기 꺼내기        │
│                                         │
│  P — 진단 PDF 꺼내서 숫자 보여주기        │
│      → 경쟁사 리뷰 수 vs 사장님 리뷰 수   │
│                                         │
│  I — "이 순위면 한 달에 [X]명 놓치고      │
│       계세요" 숫자로 환산                 │
│                                         │
│  N — "이거 고치면 [X]명 더 올 수 있어요" │
│      → 패키지 제안 (C→B→A 순서)          │
│                                         │
├─────────────────────────────────────────┤
│  클로징 기준                             │
│  "화/목 중에 언제 계약서 갖고 올까요?"    │
├─────────────────────────────────────────┤
│  절대 금지                              │
│  - 가격 즉시 깎기                       │
│  - "일단 무료로 해드릴게요"              │
│  - 매출 보장 약속                       │
└─────────────────────────────────────────┘
```

## 패키지 한눈에 — 현장 암기용

```
A. 플레이스 케어    → 38만원/월 (3개월~)
   : 사진/소식/리뷰응대/월리포트

B. 바이럴 성장     → 56만원/월 (3개월~)
   : A 전부 + 블로그체험단 2건 + 리워드트래픽
     + 영수증리뷰이벤트 + 카카오채널

C. 광고 풀케어     → 95만원/월 + 광고비별도 (3개월~)
   : B 전부 + 인스타관리 + 메타/당근광고
     + 커뮤니티침투 + 전담담당자
```
