|------|---------|-----------|

| 영수증 리뷰 이벤트 | 리뷰 인센티브 제공 시 네이버 정책 위반 가능 | "적발 시 광고주 책임, 에이전시 귀책 없음" |
| 의료 업종 블로그 체험단 | 심의 서류 처리 대행 명시 필수 | 심의 완료 후 진행 조건 명시 |

## Lead Magnet: 무료 플레이스 진단

- naver-diagnosis 툴 → 업체명 입력 → 진단 리포트 PDF 자동 생성
- 영업 첫 DM: "진단 결과 보내드릴게요" → 계약 입구
- 비용: 0원 / 소요 시간: 3분
- 전환 목표: 진단 받은 업체 → 패키지 계약 전환율 20% 이상

## 패키지 A — 플레이스 기본 케어
> "검색 되게 만들고, 리뷰 관리 시작하자"

**가격**: 월 380,000원 (VAT 별도) / 최소 3개월
**타겟**: 플레이스 사진 10장 이하 / 리뷰 응대율 30% 미만 / 소식 0개 업체

### 구성
| 항목 | 내용 |
|------|------|
| N플레이스 정보 최적화 | 사진 정리, 영업시간, 소개글, 메뉴 정비 (월 1회 점검) |
| 소식 등록 | 월 3건 (이벤트/계절 메뉴/공지) |
| 리뷰 사장님 응대 | 신규 리뷰 대신 작성 (월 최대 15건) |
| 진단 리포트 → 월 리포트 | 초기 점수 + 매월 변화 추적 |

### 원가 구조
| 항목 | 월 원가 |
|------|--------|
| 운영 인건비 (4시간/월 × 20,000) | 80,000원 |
| 리포트 작성 인건비 | 20,000원 |
| **합계** | **100,000원** |

- 마진율: **약 74%** (380,000 - 100,000 = 280,000원)

### 영업 포인트
- "경쟁사 리뷰 응대율 89% vs 지금 12% — 손님들이 리뷰 달아도 답장도 없어요"
- 진단 리포트 시각화로 "이게 안 되어 있어요" 즉시 증명 가능

## 패키지 B — 바이럴 성장
> "리뷰 쌓고, 검색 유입 늘리자"

**가격**: 월 560,000원 (VAT 별도) / 최소 3개월
**타겟**: 리뷰 50개 이하, 유입 늘리고 싶은 맛집·카페·미용실 (의료 업종 제외)
**업셀 경로**: 패키지 A → 2개월차에 제안

### 구성
| 항목 | 내용 |
|------|------|
| 플레이스 기본 케어 전부 | 패키지 A 구성 전부 |
| 블로그 체험단 2건/월 | 방문형 (플랫폼 섭외 대행, 뒷광고 표시 포함) |
| 리워드 트래픽 | 플레이스 방문 유도 (면책 조항 포함, 월 예산 고객 협의) |
| 영수증 리뷰 이벤트 | 방문 고객 대상 리뷰 유도 이벤트 설계 (면책 조항 포함) |
| 카카오채널 관리 | 채널 세팅 + 메시지 발송 2건/월 |
| 성과 리포트 | 플레이스 순위 변화 + 리뷰 수 추이 포함 |

### 원가 구조
| 항목 | 월 원가 |
|------|--------|
| 패키지 A 원가 | 100,000원 |
| 블로그 체험단 2건 × 15,000 | 30,000원 |
| 리워드 트래픽 운영비 | 20,000원 |
| 영수증 리뷰 이벤트 기획 | 10,000원 |
| 카카오채널 운영 인건비 (2시간) | 40,000원 |
| **합계** | **200,000원** |

- 마진율: **약 64%** (560,000 - 200,000 = 360,000원)

### 영업 포인트
- "경쟁사 리뷰 237개 vs 지금 18개 — 3개월이면 50개 이상 만들 수 있어요"
- 체험단 = 광고 표시 의무 있음 → 계약서에 뒷광고 위반 시 광고주 책임 명시

## 패키지 C — 광고 풀케어
> "신규 고객 광고로 직접 끌어오자"

**가격**: 월 950,000원 + 광고비 별도 (VAT 별도) / 최소 3개월
**타겟**: 월 매출 500만원 이상, 광고 의향 있는 업체
**광고비 권장**: 메타 기준 월 30~80만원 별도 (고객이 직접 결제)

### 구성
| 항목 | 내용 |
|------|------|
| 바이럴 성장 전부 | 패키지 B 구성 전부 |
| SNS 관리 | 인스타그램 피드 4건/월 + 스토리 8건/월 |
| 메타 or 당근 광고 운영 | 캠페인 세팅 + 월 운영 + 소재 제작 지원 |
| 커뮤니티 침투 | 지역 맘카페 or 당근 커뮤니티 홍보 2건/월 |
| 월 전략 미팅 | 30분 전화/카카오 (다음 달 방향 논의) |
| 전담 담당자 배정 | 문의 당일 응대 |

### 원가 구조
| 항목 | 월 원가 |
|------|--------|
| 패키지 B 원가 | 170,000원 |
| SNS 관리 인건비 (4시간) | 80,000원 |
| 광고 운영 인건비 (6시간) | 120,000원 |
| 소재 제작 (템플릿 기반) | 30,000원 |
| **합계** | **400,000원** |

- 마진율: **약 58%** (950,000 - 400,000 = 550,000원)

### 영업 포인트
- "광고비는 사장님이 직접 결제, 저희는 운영만 — 투명한 구조"
- 메타 기준 월 50만원 광고비로 인스타 팔로워 500~1,000명 도달 가능

## 업종별 추천 패키지

| 업종 | 1순위 | 이유 |
|------|-------|------|
| 미용실 (원장 혼자) | B → C | 리뷰 + 플레이스가 예약 직결 |
| 음식점/카페 | A → B | 먼저 검색 되게, 이후 리뷰 쌓기 |
| 학원 | B + C | 플레이스 + 메타 광고가 학부모 유입에 효과 |
| 네일/피부/마사지 | B → C | 인스타 기반, 광고로 확장 |
| 의료 (피부과·치과) | C (체험단 제외) | 체험단 심의 복잡 → 광고+SNS만 |

## 성장 경로 (고객 LTV 설계)

```
무료 진단
    ↓ (전환율 목표 20%)
A. 플레이스 케어 (월 38만, 3개월)
    ↓ (2개월차 제안, 전환율 목표 40%)
B. 바이럴 성장 (월 56만, 3개월)
    ↓ (3개월 후 제안, 전환율 목표 30%)
C. 광고 풀케어 (월 95만, 6개월+)
```

**고객 1인 평균 LTV (12개월)**
- A × 3개월 → B × 3개월 → C × 6개월
- = (380,000 × 3) + (560,000 × 3) + (950,000 × 6)
- = 1,140,000 + 1,680,000 + 5,700,000
- = **약 852만원/년 (1인)**
