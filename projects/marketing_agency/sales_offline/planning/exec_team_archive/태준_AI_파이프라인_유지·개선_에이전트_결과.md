## AI 파이프라인 운영 파트 (태준 담당)

**작성자:** 태준 (AI 파이프라인 유지·개선 에이전트) | **연계:** 서연 콘텐츠 실행안 + 민재 광고 운영안 + 지은 성과 보고안 후속 | **작성일:** 2025년 6월

---

> 📌 **태준 파트 역할 정의**
> 서연이 콘텐츠를 만들고, 민재가 광고를 돌리고, 지은이 보고하면 — **태준은 이 모든 작업이 사람 손 없이 돌아가도록 파이프라인을 설계·유지·개선하는 역할**입니다.
> 콘텐츠 원가율 40% 이하 유지가 핵심 목표이며, 반복 작업을 자동화해 팀 전체 운영 효율을 높입니다.

## SECTION 1. 헤어림 콘텐츠 원가율 현황 진단

### 1-1. 원가율 계산 기준

```
콘텐츠 원가율 = (콘텐츠 제작 투입 비용) ÷ (해당 고객 월 수수료) × 100

투입 비용 = AI 도구 사용료(고객 배분) + 담당자 작업 시간 × 시간당 단가 + 외부 리소스
```

### 1-2. 헤어림 초기 원가율 추정 (계약 전 기준)

| 항목 | 월 투입량 | 단가 | 월 비용 |
|------|----------|------|---------|
| 블로그 포스팅 | 8편 | 담당자 30분/편 × 시급 15,000원 | 60,000원 |
| 플레이스 사진 캡션 | 20개 | 담당자 10분/개 × 시급 15,000원 | 50,000원 |
| 리뷰 답글 작성 | 월 20건 | 담당자 5분/건 × 시급 15,000원 | 25,000원 |
| AI 도구 사용료 (배분) | — | ChatGPT Plus 월 2만원 ÷ 고객 10개사 | 2,000원 |
| **합계** | | | **137,000원** |

> **가정: 헤어림 월 수수료 30만 원 기준**
> 현재 추정 원가율: 137,000 ÷ 300,000 = **45.7%** → ⚠️ 목표치(40%) 초과

**즉시 자동화 개입이 필요한 상태입니다.**

### 2-2. 1순위 자동화 파이프라인 — 블로그 포스팅 초안 자동 생성

**현재 흐름 (자동화 전):**
```
담당자가 주제 선정 → 직접 키워드 조사 → 직접 초안 작성 (30분/편) → 검수 → 발행
```

**자동화 후 흐름:**
```
[Make] 구글 시트 트리거 (주제 입력)
→ [ChatGPT API] 프롬프트 자동 실행 → 초안 생성 (3분)
→ [Make] 구글 독스에 자동 저장
→ 담당자 검수 알림 (슬랙/카카오)
→ 담당자 최종 확인 (10분) → 발행
```

**적용 프롬프트 (즉시 사용 가능):**

```
당신은 양주 헤어림 미용실의 네이버 블로그 콘텐츠 작성 전문가입니다.

[고객 정보]
- 상호명: 헤어림
- 위치: 경기도 양주시
- 핵심 서비스: [주제에 따라 입력]
- 타깃: 양주 거주 30~50대 여성

[작성 조건]
- 분량: 800~1,000자
- 키워드 자연 포함: "양주 미용실", "양주 헤어", [서비스명]
- 첫 문단: 고객 고민/상황으로 시작 (공감형)
- 중간: 헤어림의 구체적 해결책 서술
- 마지막 문단: 네이버 플레이스 예약 유도 CTA 포함
- 말투: 친근하되 전문적, 광고 티 나지 않게

[이번 주제]: {{주제}}

위 조건에 맞춰 블로그 포스팅 초안을 작성해주세요.
```

**개선 효과:**

| 항목 | 자동화 전 | 자동화 후 | 절감 |
|------|----------|----------|------|
| 편당 소요 시간 | 30분 | 10분 (검수만) | 20분 절감 |
| 월 8편 기준 | 240분 | 80분 | **160분 절감** |
| 월 비용 | 60,000원 | 20,000원 | **40,000원 절감** |

### 2-3. 1순위 자동화 파이프라인 — 리뷰 답글 자동 초안 생성

**현재 흐름 (자동화 전):**
```
담당자가 리뷰 직접 확인 → 개별 답글 작성 (5분/건) → 직접 등록
```

**자동화 후 흐름:**
```
[Make] 네이버 플레이스 리뷰 RSS 또는 웹훅 감지
→ 리뷰 텍스트 추출
→ [ChatGPT API] 답글 초안 자동 생성
→ [구글 시트] 자동 정리 (리뷰 원문 + 초안 답글)
→ 담당자 일괄 검토 (주 2회, 30분) → 복붙 등록
```

**적용 프롬프트 (즉시 사용 가능):**

```
당신은 양주 헤어림 미용실 사장님입니다.
아래 고객 리뷰에 대한 네이버 플레이스 답글을 작성해주세요.

[답글 작성 원칙]
- 분량: 3~5문장
- 말투: 따뜻하고 진심 어린 감사 표현
- 리뷰에서 언급된 구체적 내용(시술명, 만족 포인트) 반드시 언급
- 재방문 유도 문구 자연스럽게 포함
- "감사합니다"로 끝내지 말고 다음 방문 기대 표현으로 마무리
- 과장 표현, 이모티콘 과다 사용 금지

[고객 리뷰]: {{리뷰 원문}}

답글을 작성해주세요.
```

**개선 효과:**

| 항목 | 자동화 전 | 자동화 후 | 절감 |
|------|----------|----------|------|
| 건당 소요 시간 | 5분 | 1분 (확인·복붙) | 4분 절감 |
| 월 20건 기준 | 100분 | 20분 | **80분 절감** |
| 월 비용 | 25,000원 | 5,000원 | **20,000원 절감** |

### 2-4. 2순위 자동화 — 성과 데이터 자동 수집 및 지은 보고서 연동

**설계:**
```
[Make] 매주 월요일 09:00 자동 실행
→ 네이버 플레이스 통계 페이지 스크래핑 (조회수, 저장수, 전화 클릭수)
→ [구글 시트] 주간 지표 자동 입력
→ [지은 보고서 템플릿] 자동 수치 채우기
→ 지은에게 슬랙 알림 → 멘트만 추가 후 발송
```

> 이 파이프라인이 완성되면 지은의 월간 보고서 작성 시간이 **3시간 → 30분
