## SECTION 2. 현황 진단 — 광고 관점

### 2-1. 왜 광고가 필요한가

준혁 에이전트 진단 기준, 헤어림은 현재 **'양주 미용실' 검색 15위권 밖**입니다.
서연 에이전트 콘텐츠 작업(플레이스 정보 보강 + 리뷰 누적)은 **자연 상승에 3~6개월** 소요됩니다.

**광고는 그 공백을 메우는 수단입니다. 둘은 경쟁 관계가 아닙니다.**

| 구분 | 네이버 플레이스 자연 최적화 | 네이버 광고 (검색·플레이스) |
|------|--------------------------|--------------------------|
| 효과 발생 시점 | 3~6개월 | 집행 즉시 |
| 비용 | 인건비·콘텐츠 비용 | 클릭당 과금 (CPC) |
| 지속성 | 장기 자산 축적 | 예산 소진 시 중단 |
| 역할 | 신뢰도·기반 구축 | 즉시 유입·예약 확보 |

> **결론:** 콘텐츠 최적화 기간 동안 광고로 유입을 확보하고, 자연 순위가 오르면 광고비를 점진적으로 줄이는 구조가 최적입니다.

> ※ 아래는 **월 예산 확정 전 시나리오**입니다. 계약 시 1개 플랜으로 확정합니다.

| 항목 | 내용 |

## SECTION 4. 키워드 전략 (네이버 검색광고 기준)

### 4-1. 핵심 키워드 세트

| 키워드 유형 | 키워드 예시 | 입찰가 기준 | 우선순위 |
|------------|-----------|-----------|---------|
| 지역 메인 | 양주 미용실, 양주시 미용실 | 700~1,200원 | ★★★ |
| 지역 세부 | 양주 헤어, 양주 커트, 양주 염색 | 400~800원 | ★★★ |
| 시술 특화 | 양주 매직, 양주 펌, 양주 클리닉 | 300~600원 | ★★☆ |
| 경쟁 우회 | 양주역 미용실, 덕계 미용실 | 200~400원 | ★★☆ |
| 롱테일 | 양주 여성 헤어, 양주 가성비 미용실 | 150~300원 | ★☆☆ |

> **집행 원칙:** 초기 2주는 전체 키워드 집행 → 클릭·전환 데이터 확인 후 성과 하위 30% 키워드 제거, 상위 키워드 입찰가 증액

```
서울 미용실 / 의정부 미용실 / 일산 미용실
남성 전용 / 바버샵 / 이발소
(온보딩 인터뷰 후 시술 범위 확인하여 추가 설정)
```

## SECTION 5. 광고 소재 초안

> ⚠️ 서연 에이전트 온보딩 인터뷰 완료 전까지 **확정 불가** — 아래는 구조 초안입니다.

### 5-1. 네이버 검색광고 텍스트 소재

**소재 A (가격 소구)**
```
제목: 양주 미용실 헤어림 | [대표시술] [가격]원~
설명: 양주 [경력]년 헤어 전문 | 당일 예약 가능 | 지금 전화하세요
```

**소재 B (전문성 소구)**
```
제목: 양주 헤어 전문 헤어림 | [전문분야] 특화 원장
설명: [수상·자격] 보유 | [운영시간] 운영 | 네이버 예약 바로 가능
```

**소재 C (리뷰·신뢰 소구)** ← 리뷰 50개 이상 달성 후 활성화
```
제목: 양주 미용실 헤어림 | 리뷰 [N]개 지역 인기
설명: [대표 리뷰 문구 한 줄] | 주차 가능 | 예약 우선 안내
```

> **소재 교체 기준:** CTR이 **업종 평균(1.5%) 대비 50% 이하(0.75% 미만)** 시 즉시 교체

|----------|------|------|
