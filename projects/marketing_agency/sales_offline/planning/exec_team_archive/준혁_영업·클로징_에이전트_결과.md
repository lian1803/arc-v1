### 💰 2-1. 월 기회손실 추정

```
양주시 '미용실' 관련 네이버 월 검색량: 약 3,200회
상위 1~3위 클릭 집중률: 전체의 68%
→ 상위권 점포는 월 약 2,176클릭 확보

현재 헤어림 추정 클릭: 월 80~120회 (하위 노출)

상위권 진입 시 추가 확보 가능 클릭: 약 +1,500회/월
플레이스→예약 전환율 평균: 8~12%
→ 월 추가 신규 고객 유입 가능: 약 120~180명

헤어컷 1회 평균 객단가: 35,000원 기준
→ 월 기회손실: 약 420만 ~ 630만 원
→ 연간 기회손실: 약 5,000만 ~ 7,500만 원
```

> **매달 500만 원짜리 간판이 꺼져있는 상태입니다.**

## 🛠️ SECTION 3. 제안 패키지 — 헤어림 맞춤 3단계 플랜

---

### PLAN A — 플레이스 기초 정비 *(즉시 출혈 막기)*

**월 99,000원 | 3개월 약정**

| 실행 항목 | 세부 내용 |
|-----------|---------|
| 플레이스 프로필 최적화 | 키워드 재배치, 서비스 항목 세분화, 해시태그 20개 |
| 사진 콘텐츠 정비 | 전문 사진 촬영 가이드 제공 + 업로드 대행 (30장) |
| 영업정보 완성 | 공휴일·임시휴무·주차 등 세부 정보 완성 |
| 리뷰 유도 스크립트 | 고객 자연 리뷰 유도 문자·카카오 템플릿 제공 |
| 월 1회 현황 리포트 | 순위 변동, 클릭 수, 리뷰 증가 수치 보고 |

**예상 효과 (3개월):** 리뷰 +30~50개, 검색 순위 10위권 진입

---

### PLAN B — 플레이스 + 블로그 복합 공략 *(추천 ⭐)*

**월 249,000원 | 3개월 약정**

> PLAN A 전체 포함 +

| 추가 실행 항목 | 세부 내용 |
|--------------|---------|
| 네이버 블로그 리뷰 월 4건 | 지역 키워드 최적화 포스팅 ('양주 펌 잘하는 곳' 등) |
| 플레이스 Q&A 관리 | 자주 묻는 질문 등록 + 답변 최적화 |
| 경쟁사 키워드 역이용 | 경쟁사 검색 시 헤어림 노출 키워드 설계 |
| 월 2회 현황 리포트 | 키워드 순위 + 유입 경로 분석 |

**예상 효과 (3개월):** 상위 5위권 진입, 신규 고객 월 +40~80명 증가

---

### PLAN C — 풀 마케팅 통합 관리 *(공격적 점유율 확보)*

**월 560,000원 | 6개월 약정**

> PLAN B 전체 포함 +

| 추가 실행 항목 | 세부 내용 |
|--------------|---------|
| 네이버 스마트스토어 예약 연동 | 온라인 예약 시스템 셋업 |
| 인스타그램 콘텐츠 연동 | 월 8포스팅 피드 제작 + 플레이스 연계 |
| 광고비 운용 컨설팅 | 파워링크 or 플레이스 광고 최적 셋업 |
| 월 4회 현황 리포트 + 전략 미팅 | 매월 30분 화상 전략 미팅 |

**예상 효과 (6개월):** 상위 1~3위 진입, 신규 고객 월 +100~180명

### 📅 타이밍 논리

```
현재 양주 지역 미용실 네이버 플레이스 경쟁 상황:
- 상위 3곳: 이미 리뷰 150개+ 확보, 매월 7~11건 블로그 누적 중
