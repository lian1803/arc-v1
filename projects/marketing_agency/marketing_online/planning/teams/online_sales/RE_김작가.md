# 김작가 — 네이버 플레이스 최적화 서비스 아웃리치 시퀀스 풀 스크립트

---

## ⚠️ 사전 공지 (법적 필수 사항)

**이메일 발송 전 반드시 확인:**
- 광고성 이메일에는 제목에 **[광고]** 표시 필수 (정보통신망법 제50조)
- 본문 하단에 **수신거부 링크** 필수 포함
- 사전 수신동의 없는 대량 발송은 불법 — 수신동의 획득 프로세스 먼저 설계할 것
- 인스타그램 DM: **수동 발송 또는 Meta 공식 API 파트너 도구만 사용** (Jarvee 등 비공식 자동화 도구 사용 시 계정 영구 정지 위험)

## 📌 변수 정의표 (매 아웃리치 전 이 표를 먼저 채운다)

| 변수 | 설명 | 예시 |
|------|------|------|
| `[업체명]` | 상호명 | 강남왕돈까스 |
| `[업종]` | 업종 카테고리 | 음식점(돈까스) |
| `[지역]` | 운영 지역 | 서울 강남구 |
| `[핵심문제]` | 진단에서 발견한 가장 큰 문제 | 플레이스 리뷰 23개·경쟁사 평균 187개 |
| `[진단점수]` | 이진단 보고서 총점 | 42/100점 |
| `[경쟁사명]` | 같은 상권 상위 노출 경쟁업체 | 강남원조돈까스 |
| `[손실추정액]` | 월 기회손실 금액 (이진단 산출) | 월 180만 원 |
| `[담당자명]` | 수신인 이름 또는 사장님 | 김사장님 |

```
D+0   첫 접촉     — 문제를 건드린다 (Problem)

D+7   가치 제공   — 변화한 동종 사례로 가능성을 보여준다 (Solve)
D+14  마지막 제안 — 지금 결정하면 얻는 것, 안 하면 잃는 것
```

# D+0 | 첫 접촉 — "지금 손님이 새고 있습니다"

## 📱 인스타그램 DM 버전

> **발송 조건**: 해당 업체 인스타그램 계정에 팔로우 후 24시간 뒤 발송. 수동 발송 원칙.
> **발송 시간**: 평일 오전 10~11시 또는 저녁 8~9시
> **길이 제한**: 첫 메시지 100자 이내 → 답장 오면 2번째 메시지 발송

---

### 📩 DM 1차 메시지 (100자 이내)

```
[담당자명]님, [업체명] [업종] 운영하시죠?
네이버 플레이스 검색해봤는데 [핵심문제] 상태였어요.
이 상태면 지금 월 [손실추정액] 넘게 기회손실 나고 있을 가능성 높아요.
30초 확인 가능하세요?
```

**[PAS 주석 — Problem]**: 업체명·업종 언급으로 "나를 직접 봤다"는 개인화 신호 → 핵심문제 수치 직격 → 손실액 제시로 즉각 위기감 형성

---

### 📩 DM 2차 메시지 (답장 왔을 때 발송)

```
감사합니다 [담당자명]님!

제가 [업체명] 플레이스 간단히 살펴봤는데요.

현재 상황:
• 리뷰 수: [핵심문제 수치] → 상권 내 상위 3개 업체 평균 [경쟁사 평균 리뷰 수]개
• 플레이스 노출 순위: [현재 순위]위 (검색어: [대표 검색 키워드])
• [경쟁사명]은 이미 상단 고정 상태예요

솔직히 말씀드리면,
지금 [업종] 찾는 손님들이 [경쟁사명]한테 가고 있어요.

무료 진단서 하나 만들어서 보내드릴게요.
구체적으로 어디서 손님이 새는지 숫자로 보여드릴 수 있어요.

이메일 주소 알려주시면 오늘 중으로 보내드릴게요 🙏
```

**[PAS 주석 — Agitate]**: 경쟁사 대비 구체 수치로 현실 직시 → "손님이 경쟁사한테 가고 있다"는 긴박감 → 진단서 제공으로 자연스럽게 이메일 획득
