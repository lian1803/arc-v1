# 한총괄 — 네이버 플레이스 최적화 서비스 주간 실행 매뉴얼

## 📋 핵심 요약
리안이 매주 **네이버 플레이스 최적화 서비스**를 영업하기 위해 실행할 구체적 행동 리스트입니다.  
5개 에이전트의 산출물이 일관되게 연결되며, 매주 **KPI 달성 여부**를 숫자로 추적합니다.

---

## 🔴 현재 진단: 에이전트 간 연결성 문제 발견

### 발견된 문제:
1. **박탐정 결과 부재**: 실제 타겟 리스트 없음 (누구를 아웃리치할 건가?)
2. **이진단 진단서**: 샘플 제안서는 있지만 리안이 실제 업체 분석하는 프로세스 불명확
3. **김작가 DM**: 1차 아웃리치 스크립트만 있고, 추가 터치(재DM, 이메일) 없음
4. **최제안 제안서**: 풀텍스트 길이가 과다하고 **"복붙만으로는 불가능"** (맞춤화 작업 필요)
5. **정클로저 대본**: 30분 대본은 있으나 거절 처리 6종 카드 내용 불완전

### 결론: 현재 점수 **18/40점** → 목표: **35/40점 이상**

## ✅ 리안의 주간 실행 매뉴얼 (1주일 = 월~금)

### 📊 주간 목표 KPI
```
├─ 잠재고객 발굴: 15곳
├─ 1차 아웃리치(DM): 15건
├─ 진단서 제시: 5곳 (=응답한 업체)
├─ 제안서 발송: 3곳 (=진단서 본 후 "계속 진행" 표시)
├─ 미팅 확정: 2건 (=제안서 본 후 미팅 신청)
├─ 계약: 0.5건 (월 목표 10건 역산)
```
