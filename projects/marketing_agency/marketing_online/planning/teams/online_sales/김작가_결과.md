# 김작가 — 네이버 플레이스 최적화 서비스
## 4단계 아웃리치 시퀀스 풀 스크립트 v3.0

---

> **⚠️ RISE 반성 (이전 실행 실패 원인 → 이번 개선 포인트)**
>
> **실패 1: 대상 혼동** → 쿠팡/스마트스토어 셀러 프레임으로 소상공인에게 DM 씀. 이번엔 오프라인 매장 소상공인(음식점·미용실·학원·카페 등) 전용으로 완전 재설계
>
> **실패 2: 차별화 없음** → "네이버 플레이스 최적화 어떠세요?" 수준의 범용 멘트. 이번엔 업체 진단 수치(리뷰 수·검색 순위·손실 추정액)를 첫 문장에 박아넣음
>
> **실패 3: 고객 이탈 가능성** → 스크립트가 너무 길고 영업 냄새가 남. 이번엔 D+0은 2문장, 각 단계 분리, 서사 흐름(문제→공감→해결)을 심리 단계별로 설계
>
> **이번 목표**: 김작가 자가평가 8점 이상 — "나였으면 답장할 것 같은가?"

```

D+7  ─── 가치 제공   : 동종 성공 사례 공유 → 변화 가능성 제시 (Agitate→Solve)
D+14 ─── 마지막 제안 : 한정 혜택 + 클로징 → 결정 촉구 (Solve)
```

## 📱 D+0 — 인스타그램 DM 버전

```
──────────────────────────────────────────────────
[업체명] 운영하시는 [담당자명],
혹시 [지역] 네이버 검색하면 지금 몇 위로 뜨시는지 아세요?

오늘 직접 확인해봤는데,
[핵심문제] 상태더라고요.

바로 위 경쟁사([경쟁사명])는 리뷰 300개 넘게 쌓여있고요.

30초짜리 무료 진단 결과 PDf 드려도 될까요?
보시고 도움 안 되면 그냥 버리셔도 됩니다.
──────────────────────────────────────────────────
```

**[PAS 주석]**
- **P(Problem)**: "지금 몇 위로 뜨는지 아세요?" → 현재 상황을 모른다는 불안감 자극
- **A(Agitate)**: [핵심문제] + 경쟁사 수치 병기 → 손실이 실제로 존재함을 각인
- **S(Solve 예고)**: "30초짜리 무료 진단" → 부담 없는 다음 단계 제시

**발송 운영 지침**
```
발송 방식 : 수동 1:1 DM (Meta 공식 정책 준수)
최적 시간 : 평일 오전 10:00~11:00 / 저녁 20:00~21:00
개인화 필수: [업체명] [담당자명] [지역] [핵심문제] 반드시 실제 수치로 교체
금지 사항 : 동일 계정에서 하루 20건 초과 DM 금지 (계정 제한 위험)
```

**[PAS 주석]**
- **P(Problem)**: 검색 결과 수치 3줄 — 스캔만 해도 현재 문제 인식

- **S(Solve 예고)**: "진단서 무료" + "네 한 마디" → 마찰 제거, 다음 단계 연결
