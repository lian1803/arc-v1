# 히디프(HIDIFF) 맞춤형 콘텐츠 전략 덱
### — 서연 (콘텐츠 전략가) | 인스타D2C마케팅팀

---

## ⚡ 전략가 한 줄 진단

> **"히디프는 '좋은 제품을 가진 브랜드'가 아니라 '증명된 브랜드'입니다. 화해 3관왕이라는 신뢰 자산이 있는데, 콘텐츠가 그걸 한 번도 제대로 팔지 않았습니다."**

팔로워 2만 명 + 수상 이력 + 창업자 스토리. 이 세 가지가 동시에 있는 브랜드는 흔하지 않습니다.
문제는 **이 자산들이 각각 따로 존재**하고 있다는 것. 지금 필요한 건 연결입니다.

## PART 1. 브랜드 포지셔닝 재정의

### 현재 포지셔닝 (추정)
```
"피부에 좋은 스킨케어 브랜드"
→ 카테고리 내 수천 개 브랜드와 구분 불가
```

### 제안 포지셔닝
```
"검증된 성분, 검증된 결과 — 화해가 인정한 스킨케어"
→ 수상 이력을 상시 포지셔닝 무기로 전환
```

### 타깃 페르소나 (1순위)
| 항목 | 정의 |
|------|------|
| **연령/성별** | 25~35세 여성 |
| **핵심 고민** | "성분은 공부했는데 뭘 써야 할지 모르겠다" |
| **정보 수집 방식** | 화해 앱 후기 → 인스타 탐색 → 구매 결정 |
| **구매 결정 요소** | 실사용 후기 + 성분 신뢰 + 브랜드 진정성 |
| **콘텐츠 반응 패턴** | 저장/공유 중심 (즉각 좋아요보다 나중에 사러 옴) |

> **전략적 함의**: 이 타깃은 **감성보다 근거**에 반응합니다. 예쁜 피드보다 "왜 이 제품인지"를 설명하는 콘텐츠가 전환율을 올립니다.

## PART 2. 콘텐츠 필러 5개 (브랜드 반복 패턴 설계)

> 필러(Pillar)란: 매주/매월 반복 발행하는 콘텐츠 기둥. 브랜드 정체성을 형성하고 알고리즘 신호를 강화합니다.

---

### 🧪 필러 1. "성분 교실" — 신뢰 구축

**목적**: 전문성 포지셔닝 + 저장률 극대화
**포맷**: 캐러셀 (7~10장)
**발행 주기**: 주 1회 (화요일)
**콘텐츠 구조**:

```
슬라이드 1: 훅 — "피부과 원장님이 나이아신아마이드를 쓰는 이유"
슬라이드 2~6: 성분 설명 (히디프 제품 성분 연결)
슬라이드 7~9: 히디프 제품에서 이 성분을 어떻게 썼는지
슬라이드 10: CTA — "저장하고 성분 체크하세요"
```

**KPI**: 저장률 5% 이상 (업계 평균 저장률 1~2% 대비)
**실제 레퍼런스**: @glow_recipe (성분 설명 캐러셀) → 저장 수 좋아요의 3~5배

---

### 🏆 필러 2. "수상의 이유" — 증거 기반 설득

**목적**: 화해 3관왕 자산을 상시 전환 도구로 활용
**포맷**: 릴스 (30~45초) + 캐러셀 교차
**발행 주기**: 월 2회
**콘텐츠 구조**:

```
[릴스 버전]
0~3초: "화해 앱에서 수천 명이 선택한 이유가 있습니다"
3~20초: 실사용자 Before/After 또는 텍스트 후기 모음
20~35초: 수상 인증 장면 + 제품 클로즈업
35~45초: CTA — "링크 인 바이오"

[캐러셀 버전]
1장: 수상 배너 (화해 공식 이미지 활용)
2~4장: 수상 제품 성분 근거
5~6장: 실사용자 후기 캡처 (동의 획득 후)
7장: 구매 유도 CTA
```

**KPI**: 해당 게시물 발행 후 72시간 내 링크 클릭률 3% 이상
**핵심 인사이트**: 수상 콘텐츠는 "자랑"이 아니라 "이유 설명"으로 프레이밍해야 저항 없이 소비됩니다.

---

### 👩‍💼 필러 3. "창업자의 노트" — 팬덤 형성

**목적**: @hyu.rim 의존 구조를 공식 계정으로 흡수
**포맷**: 세로형 릴스 (45~60초) + 정적 이미지
**발행 주기**: 주 1~2회
**콘텐츠 구조**:

```
[릴스 버전 — 브이로그 느낌]
- "오늘 R&D 미팅에서 거절한 성분이 있어요" (의사결정 스토리)
- "이 제품 출시하는 데 왜 18개월 걸렸냐면" (개발 비하인드)
- "화해 3관왕 받던 날 저는 울었습니다" (감정적 터치포인트)

[정적 이미지 버전]
- 개발 노트 사진 + 짧은 캡션
- "오늘의 성분 고민: [성분명]을 넣을까, 뺄까"
```

**KPI**: 팔로워 저장 + 댓글 반응률 (댓글 수 기존 대비 2배 이상)
**전략적 이유**: 창업자 스토리는 **유일한 차별화 요소**. 어떤 대기업도 복제 불가. 이걸 공식 계정으로 이전하면 개인 계정 의존 리스크가 해소됩니다.

---

### 🔄 필러 4. "루틴 설계" — 구매 직결 전환 콘텐츠

**목적**: 팔로워를 구매자로 전환하는 직접 퍼널
**포맷**: 캐러셀 + 릴스 교차
**발행 주기**: 주 1회 (목요일)
**콘텐츠 구조**:

```
[피부 타입별 루틴 캐러셀]
1장: "건성 피부 주 1회 집중 루틴" (훅)
2~5장: Step-by-step 루틴 (히디프 제품 자연스럽게 포함)
6장: "이 루틴에 사용된 제품" + 링크 안내
7장: "내 피부 타입 모르면? 댓글로 알려주세요"

[계절/상황별 변형]
- "환절기 3일 집중 루틴"
- "출장 전날 밤 응급 루틴"
- "화해 3관왕 제품만으로 완성하는 미니멀 루틴"
```

**KPI**: 게시물 → 링크 클릭 전환율 2.5% 이상, 댓글 반응 유도
**CTA 전략**: "댓글로 피부 고민 남기면 맞춤 루틴 추천해드립니다" → DM 퍼널로 연결

---

### 💬 필러 5. "진짜 후기 큐레이션" — 사회적 증거 증폭

**목적**: 기존 고객 후기를 신규 구매자 설득 도구로 전환
**포맷**: 스토리 + 피드 캐러셀
**발행 주기**: 주 2회 (스토리 상시 / 피드 격주)
**콘텐츠 구조**:

```
[스토리 활용]
- 화해/네이버/인스타 후기 캡처 공유
- 투표 기능: "이 고민 나도 있어요?" → 참여율↑
- 스와이프업(또는 링크 스티커)으로 구매 페이지 연결

[피드 캐러셀]
1장: 후기 핵심 문장을 크게 (ex. "3주 만에 모공이 달라졌어요")
2~4장: 후기 원문 캡처 (동의 획득 필수)
5장: "당신의 후기를 기다립니다" + 해시태그 안내
```

**KPI**: 스토리 링크 클릭률 8% 이상 (업계 평균 5%)
**운영 팁**: 후기 제공자에게 작은 혜택(적립금, 샘플)을 제공하면 후기 수집이 지속적으로 가능합니다.
