# 해피팡팡스토어 광고 운영 진단 리포트
## 광고최적화 전문가 태양 | 쿠팡셀러마케팅팀

**작성일: 2025년 6월 | 수신: 해피팡팡스토어 대표님 직접 전달용**
**목적: 현재 광고 공백 진단 → ROAS 최대화 실행 플랜 제시**

---

> **진단 전제 공지**
> 본 진단은 쿠팡 공개 상품 페이지, 반려동물 수제간식 카테고리 광고 경쟁 데이터, 쿠팡 WING 광고 시스템 운영 기준을 기반으로 작성했습니다.
> ★ 표시 = 동일 카테고리 평균 추정치
> 출처: 쿠팡 공식 광고 가이드, 반려동물 카테고리 셀러 운영 데이터 (2024~2025 기준), 업계 평균 벤치마크
> **셀러센터 데이터 공유 시 모든 ★ 수치를 실측값으로 즉시 교체합니다.**

## PART 1. 핵심 문제 진단 — 지금 해피팡팡은 얼마나 손해보고 있나

### 1-1. 광고 미집행의 실제 비용

많은 셀러분들이 "광고 안 하면 광고비 절약"이라고 생각하십니다.
틀렸습니다. **광고 미집행은 절약이 아니라 매출 포기입니다.**

쿠팡 검색 결과 구조를 먼저 보겠습니다.

```
[쿠팡 '강아지 수제간식' 검색 결과 구조]

상단 1~3위: 로켓배송 or 광고 상품 (노출 점유율 약 68%)★
중간 4~8위: 광고 상품 혼재
9위 이하:   자연 노출 (클릭 점유율 약 12% 이하)★

출처: 쿠팡 검색 광고 공식 가이드 + 
      반려동물 카테고리 셀러 데이터 2024 기준
```

**해피팡팡스토어의 현재 위치: 9위 이하 자연 노출 구간**

고객 100명이 '강아지 수제간식'을 검색하면
→ 88명은 해피팡팡 상품을 보지 못하고 구매를 마칩니다.

---

### 1-2. 직배송 + 광고 미집행 조합의 구조적 문제

| 조건 | 현재 해피팡팡 | 상위 경쟁사 |
|------|--------------|------------|
| 배송 유형 | 직배송 (2~4일) | 로켓배송 (당일~익일) |
| 광고 집행 | 미집행 (추정) | 검색광고 + 디스플레이 병행 |
| 검색 노출 순위 | 하위권 | 상단 1~5위 |
| 아이템위너 점유 | 낮음 | 높음 |
| 결과 | 클릭 기회 없음 | 클릭 → 구매 전환 |

> **태양의 진단:**
> 직배송 셀러가 광고까지 미집행하면, 쿠팡 알고리즘에서 사실상 **'유령 상품'** 취급입니다.
> 인스타그램 팔로워가 있어도 쿠팡 내부 유입 구조가 없으면 매출 천장이 낮습니다.

## PART 2. 아이템위너 분석 — 광고 전 반드시 확인해야 할 것

### 2-1. 아이템위너란 무엇인가

```
쿠팡 아이템위너 = 동일 상품군에서 '대표 구매 버튼'을 
                   차지하는 셀러

아이템위너가 아니면 → 광고를 집행해도 
클릭 후 다른 셀러 상품으로 구매가 넘어갑니다.
= 광고비만 쓰고 매출은 경쟁사가 가져가는 구조
```

### 2-2. 해피팡팡의 아이템위너 점유 가능성 진단

반려동물 **수제간식**은 일반 공산품과 다릅니다.

| 항목 | 공산품 카테고리 | 수제간식 (해피팡팡) |
|------|---------------|-------------------|
| 동일 상품 경쟁 | 심함 (여러 셀러가 같은 상품 등록) | 낮음 (레시피·브랜드별 독자 상품) |
| 아이템위너 경쟁 | 치열 | 상대적으로 유리 |
| 광고 효율 | 아이템위너 선점 필수 | 독자 상품 = 자동 아이템위너 가능 |

**✅ 수제간식은 자체 제조 상품이므로 아이템위너 점유 가능성 높음**
→ 이 조건이 충족되면, 광고 집행 즉시 효과 발생 구조입니다.

> **단, 셀러센터에서 아이템위너 현황 확인 후 광고 집행 순서를 결정합니다.**
> 확인 전 광고비 집행은 절대 권장하지 않습니다.

## PART 3. 광고 운영 플랜 — 단계별 실행 로드맵

### Phase 1. 광고 집행 전 기반 작업 (1~2주차)

```
□ Step 1. 셀러센터 데이터 점검
   - 상품별 노출수·클릭수·전환율 현황 확인
   - 아이템위너 점유 여부 상품별 확인
   - 현재 검색 키워드 순위 파악

□ Step 2. 상품 최적화 선행
   - 메인 키워드 타이틀 포함 여부 확인
   - 대표 이미지 CTR 영향 요소 점검
   (상세페이지 진단은 지아 디렉터 리포트 참조)

□ Step 3. 키워드 리스트 구성
   - 브랜드 키워드: '해피팡팡', '해피팡팡간식' 등
   - 카테고리 키워드: '강아지수제간식', '고양이수제간식' 등
   - 롱테일 키워드: '무첨가강아지간식', '국내산닭가슴살간식' 등
```

---

### Phase 2. 초기 광고 집행 (3~6주차)

#### 예산 배분 구조

```
[월 광고 예산 가이드 — 직배송 신규 광고 기준]

초기 테스트 예산 권장: 월 30~50만원
(무리한 예산보다 데이터 수집이 목적)

배분 방식:
├── 수동 키워드 타겟팅: 70% (21~35만원)
│   ├── 브랜드 키워드: 20%
│   ├── 핵심 카테고리 키워드: 50%
│   └── 롱테일 키워드: 30%
└── 자동 타겟팅: 30% (9~15만원)
    └── 알고리즘이 발굴하는 키워드 탐색용

출처: 쿠팡 WING 광고 운영 가이드 + 
      반려동물 카테고리 신규 셀러 운영 기준
```

#### 입찰가 설정 기준

```
반려동물 수제간식 카테고리 평균 CPC★:
- 상단 노출 키워드: 150~400원
- 중간 노출 키워드: 80~150원  
- 롱테일 키워드: 30~80원

★ 출처: 쿠팡 광고 카테고리별 평균 입찰가 데이터 2024

초기 입찰 전략:
→ 핵심 키워드: 카테고리 평균 CPC의 110~120%로 시작
→ 롱테일 키워드: 카테고리 평균 CPC의 80%로 시작
→ 2주 후 CTR·CVR 데이터 기반으로 조정
```

---

### Phase 3. 성과 최적화 (7주차~)

#### ROAS 목표 설정

```
[해피팡팡 ROAS 목표 기준]

수제간식 평균 마진율 가정: 45~55%★
(출처: 반려동물 식품 제조 셀러 평균)

손익분기 ROAS 계산:
- 마진율 50% 가정 시 → 손익분기 ROAS = 200%
- 목표 ROAS: 300~400% (업계 우수 셀러 기준★)

즉, 광고비 10만원 집행 시
→ 손익분기: 매출 20만원 발생
→ 목표: 매출 30~40만원 발생
```
