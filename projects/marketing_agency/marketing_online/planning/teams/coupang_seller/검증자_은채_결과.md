### 검증 1-1. "★ 수치"의 실제 신뢰도

**[현재 조건에서 사용 가능한가: 조건부YES]**

**[리스크]**
지아 리포트의 ★ 표시 수치들은 "동일 카테고리 평균 추정치"로 표기되어 있습니다.
문제는 출처가 구체적이지 않다는 것입니다.

```
현재 표기 방식: ★ = 동일 카테고리 평균 추정치 (출처 별도 명시)
실제 문제: "출처 별도 명시" → 어디에도 없음
```

대표님이 "이 수치 어디서 나왔어요?"라고 물으면 답변 불가합니다.
반려동물 카테고리 셀러 중 수치에 민감한 분들이 많습니다. 수제간식 셀러는 특히 데이터 리터러시가 높은 경우가 있습니다.
신설팀이 출처 없는 수치를 제시하다 역공 맞으면 **첫 영업에서 신뢰 0**이 됩니다.

**[현재 조건에서 보완할 것]**

★ 수치를 다음 중 하나로 즉시 처리하세요. 선택지는 두 개뿐입니다.

**선택지 A — 공개 출처로 교체 (권장)**
| 현재 수치 | 교체 출처 |
|-----------|-----------|
| 카테고리 평균 전환율 | 쿠팡 공식 WING 가이드 공개 수치 |
| 리뷰 영향도 | 오픈서베이 펫 트렌드 리포트 2024 (무료 공개) |
| 카테고리 경쟁 강도 | 쿠팡 상품 검색 결과 수 직접 캡처 (검증 가능) |

**선택지 B — 수치 삭제 후 방향만 제시**
```
수치 삭제 후 표현 예시:
"업계 평균 대비 전환율 차이가 발생하고 있으며, 셀러센터 데이터 공유 시 
정확한 수치로 진단드릴 수 있습니다."
→ 오히려 신뢰도 상승 (솔직함 = 전문성)
```

### 검증 1-2. "구매 장벽 TOP 5" 설정의 실행 가능성

**[현재 조건에서 사용 가능한가: 조건부YES]**

**[리스크]**
"쿠팡 반려동물 카테고리 리뷰 분석" 기반이라고 되어 있습니다.
그런데 이 분석을 실제로 몇 개 리뷰에서 했는지, 어떤 상품의 리뷰인지 불명확합니다.

실제 해피팡팡 상품 리뷰와 다를 수 있습니다.
예: 해피팡팡이 이미 원재료 표기를 잘 하고 있다면 → "원재료 불명확"이라는 장벽 지적은 틀린 진단이 됩니다.

**[현재 조건에서 보완할 것]**

리포트 전달 전 반드시 해피팡팡 실제 상품 리뷰 10개 직접 읽기.
10분이면 됩니다. 이것만 해도 오진 리스크 90% 차단 가능합니다.

```
확인 체크리스트 (전달 전 필수):
□ 해피팡팡 쿠팡 상품페이지 리뷰 최신 10개 확인
□ 부정 리뷰가 있다면 → 그 내용을 장벽 TOP 5에 반영
□ 긍정 리뷰에서 반복되는 키워드 → 상세페이지 강화 포인트로 활용
□ 인스타그램 댓글에서 자주 나오는 질문 → 구매 장벽 실제 데이터로 사용
```

### 검증 1-3. 상세페이지 개선안의 실행 가능성

**[현재 조건에서 사용 가능한가: NO → 조건부YES로 전환 가능]**

**[리스크]**
상세페이지 개선은 실제 실행 시 다음 조건이 필요합니다.

```
상세페이지 개선 실제 필요 조건:
① 디자이너 또는 이미지 편집 툴 (Canva 등)
② 제품 실제 사진 원본 파일 (셀러 제공 필요)
③ 성분표·원산지 증명 자료 (수제간식은 특히 민감)
④ 수정 후 쿠팡 검수 통과 시간 (최소 1~3일)
⑤ A/B 테스트 집행 시 추가 기간
```

우리 팀이 ①을 직접 수행할 수 있는지 지금 확인이 안 되어 있습니다.
"개선안을 제시"하는 것과 "우리가 직접 만들어 드린다"는 다른 약속입니다.
계약 전 영업 자료에서 이 경계가 불명확하면 → 나중에 "왜 안 해줘요?" 분쟁 가능성이 생깁니다.

**[현재 조건에서 보완할 것]**

영업 자료에 서비스 범위를 명확히 한 줄 추가하세요.

```
추가 문구 예시:
"본 진단은 개선 방향과 우선순위를 제시하며, 
실제 제작·적용은 계약 후 세부 일정을 협의하여 진행합니다."
```

이 한 줄이 나중에 분쟁을 막아줍니다.

### 검증 2-1. "광고 미집행 = 손해" 프레임의 실행 가능성

**[현재 조건에서 사용 가능한가: 조건부YES]**

**[리스크]**
"광고 안 하면 손해"라는 논리는 맞습니다. 방향은 옳습니다.
그러나 현재 리포트에서 이 손해를 수치로 환산하는 부분이 문제입니다.

광고 미집행 손실 추정 시 필요한 것:
```
① 현재 해피팡팡 월 매출 (우리가 모름)
② 카테고리 평균 광고 기여 매출 비율 (추정치만 있음)
③ 해피팡팡 상품 평균 단가 (공개 페이지에서 확인 가능 — 이건 가능)
④ 경쟁사 광고 집행 규모 (추정만 가능)
```

①이 없는 상태에서 "월 OOO만원 손실"이라고 쓰면 → 대표님이 "그 숫자 어떻게 나온 거예요?"라고 물었을 때
