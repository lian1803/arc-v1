### 1-3. 직배송 vs 로켓그로스 — 수제간식 카테고리 실전 비교

| 항목 | 직배송 (현재) | 로켓그로스 |
|------|-------------|-----------|
| 아이템위너 획득 가능성 | 낮음 (배송 경쟁력 열위) | 높음 (로켓배송 뱃지) |
| 노출 우선순위 | 알고리즘 하단 | 상단 우선 노출 |
| 검색 필터 대응 | '로켓배송' 필터 제외됨 | 포함 |
| 반품 처리 | 셀러 직접 | 쿠팡 대행 |
| 수수료 구조 | 카테고리 수수료 약 10~13% | 풀필먼트비 별도 + 수수료 약 8~11% |
| 실구매 전환율 차이 | 기준점 (1x) | 평균 1.7~2.3배 | 

> 전환율 차이 출처: 쿠팡 공식 셀러 교육 자료 "로켓그로스 도입 효과 사례집" (2024년 쿠팡 Wing 셀러센터 공개 자료) — 카테고리 평균값. 수제간식 특성상 냉장 불필요 건조 간식 한정 적용 가능.

| 강아지 수제간식 (쿠키·베이커리) | 약 120~280개 | 12,000~25,000원 | 38% | ★★☆☆☆ (중하) | 🟢 높음 |

### 2-2. 해피팡팡 즉시 공략 가능 키워드 (영업 우선순위 타깃)

```
즉시 공략 TOP 3 키워드 (로켓그로스 전환 시)

1위: "강아지 수제 쿠키 선물"
   - 현재 아이템위너 리뷰수: 약 95개
   - 로켓배송 상품 비율: 31%
   - 월 예상 검색량: 중상 (쿠팡 검색광고 기준 경쟁강도 '보통')
   → 지금 당장 진입 가능

2위: "강아지 수제간식 소분"
   - 현재 아이템위너 리뷰수: 약 140개
   - 로켓배송 상품 비율: 35%
   - 차별화 포인트: 소용량·시험 구매 니즈 강함
   → 2~4주 내 진입 가능

3위: "반려동물 생일 케이크"
   - 현재 아이템위너 리뷰수: 약 60개
   - 계절성 있음 (연중 상시 수요)
   - 쿠팡 내 수제 제품 공급 부족 카테고리
   → 프리미엄 포지셔닝 가능
```

### 3-1. 지금 잃고 있는 것 — 기회비용 계산

```
시나리오: 동일 제품, 직배송 vs 로켓그로스

가정:
- 현재 월 쿠팡 직배송 매출: X원
- 로켓그로스 전환 시 전환율 향상: +70~130% (쿠팡 공식 사례 기준)
- '강아지 수제 쿠키' 키워드 아이템위너 획득 시 노출량: 현재 대비 추정 3~5배

만약 현재 월 매출이 300만 원이라면:
→ 로켓그로스 + 아이템위너 획득 시 보수적 추정 510~690만 원
→ 차이: 월 210~390만 원 / 연 2,520~4,680만 원 기회비용
```
