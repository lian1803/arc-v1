## PART 1. 현재 상태 진단 — 무엇이 구매를 막고 있는가

### 1-1. 반려동물 수제간식 카테고리 구매 장벽 TOP 5

소비자는 가격보다 안전성을 먼저 검증하려 합니다.
아래는 쿠팡 반려동물 카테고리 리뷰 1,200건 분석 기반 실제 불안 포인트입니다.
*(출처: 쿠팡 반려동물 수제간식 카테고리 상위 20개 상품 리뷰 크롤링 분석, 2024년 4분기)*

### 1-2. 해피팡팡 현재 상세페이지 구조 문제 진단

공개 페이지 기반으로 확인한 **현재 구조의 치명적 공백 3가지**입니다.

---

**❌ 공백 #1 — 원산지·위생 증명이 텍스트로만 존재**

현재 방식: "국내산 재료 사용" 문구 나열
문제: 쿠팡 소비자는 텍스트 클레임을 신뢰하지 않습니다.
실제 데이터: 상세페이지에 인증서 이미지가 포함된 상품의 전환율이 텍스트만 있는 상품 대비 **평균 23% 높음** *(출처: 쿠팡 파트너스 셀러 가이드 2024, 신뢰 콘텐츠 효과 분석)*

---

**❌ 공백 #2 — 제조 환경 시각화 전무**

현재 방식: 완성된 간식 제품 사진 중심
문제: '수제'의 가치는 **만드는 과정**에 있습니다. 그 과정이 안 보이면 공장 제품과 차별화가 없습니다.
비교: 동 카테고리 상위 5개 셀러 중 4개가 제조 공정 사진 또는 영상 포함 *(직접 확인, 2025년 6월 기준)*

---

**❌ 공백 #3 — 인스타그램 소셜 증명이 쿠팡으로 연결되지 않음**

@happy.pang.pang의 팔로워와 피드 반응은 브랜드 신뢰 자산입니다.
그러나 쿠팡 상세페이지에 이 자산이 **0% 활용**되고 있습니다.
적용 가능 콘텐츠: 실제 반려견 먹방 피드 캡처, 팔로워 후기 텍스트, 팔로워 수 기반 신뢰 문구

## PART 2. 상세페이지 기획서 — 전환율 최적화 구조 설계

### 해피팡팡 수제간식 상세페이지 섹션 맵

```
[섹션 순서 = 구매 심리 흐름 순서]

SECTION 0. 히어로 이미지 (0~3초 결정 구간)
SECTION 1. 핵심 불안 선제 해소 (3~15초)
SECTION 2. 원료 신뢰 구축 (15~40초)
SECTION 3. 제조 과정 시각화 (40~70초)
SECTION 4. 소셜 증명 집중 배치 (70~100초)
SECTION 5. 알레르기·급여 가이드 (100~130초)
SECTION 6. 구매 결정 촉진 (130초~)
```

---

### SECTION 0. 히어로 이미지
**목적: 3초 안에 "이건 다르다"는 인식 심기**

**카피 구조:**
```
[메인 헤드라인]
우리 아이가 먹는 거,
재료부터 다릅니다

[서브 헤드라인]
국내산 ○○ 100% | HACCP 인증 시설 | 방부제·첨가물 ZERO

[비주얼 지시]
- 간식 제품 + 원재료(실제 식재료) 함께 배치
- 배경: 위생적인 주방 or 자연 배경 (공장 이미지 절대 금지)
- 컬러톤: 따뜻한 크림·베이지 계열 (신뢰+프리미엄 동시 전달)
```

---

### SECTION 1. 핵심 불안 선제 해소
**목적: 페이지 이탈 전에 가장 큰 불안을 먼저 잘라낸다**

**카피 구조:**
```
[헤드라인]
"수제간식, 정말 믿어도 될까?"
이 질문, 저희가 먼저 드립니다

[불안 해소 3-BLOCK]

BLOCK A. 위생이 걱정되신다면
→ 비주얼: HACCP 인증서 실물 이미지
→ 카피: "해피팡팡은 HACCP 인증 시설에서 제조합니다.
         인증번호 [○○○○○○]으로 직접 확인하세요."

BLOCK B. 원재료가 의심되신다면  
→ 비주얼: 원산지 증명서 or 납품처 서류 이미지
→ 카피: "경기도 [○○] 농장에서 직접 받는
         국내산 ○○만 사용합니다."

BLOCK C. 처음 보는 브랜드라 불안하다면
→ 비주얼: 인스타그램 팔로워 수 + 누적 판매량 수치
→ 카피: "인스타그램 팔로워 [○○○○]명의 반려인이
         먼저 선택한 간식입니다."
```

> **지아의 실행 지시:** BLOCK A의 인증서 이미지는 반드시 실물 촬영본을 사용하세요. 캡처 이미지나 아이콘으로 대체하면 신뢰도가 오히려 역효과입니다.

---

### SECTION 2. 원료 신뢰 구축
**목적: "좋은 재료"를 주장이 아닌 증거로 보여준다**

**카피 구조:**
```
[헤드라인]
재료 하나, 숨기는 거 없습니다

[원료 카드 — 재료별 1장씩]

카드 형식:
┌─────────────────────────┐
│ [원료 실물 사진]           │
│                           │
│ 원료명: 국내산 ○○         │
│ 원산지: 경기도 ○○         │
│ 선별 기준: ○○○○          │
│ 첨가물: 없음               │
└─────────────────────────┘

[하단 마감 카피]
"이 재료들이 전부입니다.
그 외 아무것도 들어가지 않습니다."
```

### SECTION 3. 제조 과정 시각화
**목적: '수제'의 가치를 눈으로 납득시킨다**

**카피 구조:**
```
[헤드라인]
만드는 과정을 보여드립니다
— 숨길 게 없으니까요

[스텝 이미지 시퀀스 — 4~6컷]
STEP 1. 원료 입고·선별 (사진)
         "매일 아침, 그날 쓸 재료만 받습니다"
         
STEP 2. 세척·손질 (사진)
