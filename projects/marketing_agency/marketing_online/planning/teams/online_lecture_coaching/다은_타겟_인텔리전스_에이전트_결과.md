### 🔴 최종 등급: **A등급 (최우선 타겟)**

**판정 근거 3가지:**

1. **수익 잠재력 → 상** : 26,898명 무료 수강생 × 유료 전환 2% = 537명 유료 고객 즉시 전환 가능 (업계 평균 무료→유료 전환율 1~3% 적용, 신뢰도 ★★★☆☆)
2. **성장 정체 여부 → 정체 확실** : 9년·75개 영상 = 연평균 8.3개 업로드. 구독자 대비 영상 수 극히 낮음. 콘텐츠 확장 없이 구독자 성장 한계 도달
3. **외부 마케팅 투자 의향 → 높음** : 광고 전무 + 기업교육 레퍼런스 보유 = "팔 수 있는 신뢰는 있는데 팔지 않고 있는" 전형적 구조. 한 번의 수익화 경험이 생기면 마케팅 투자로 이어지는 패턴

## 🔍 PART 2. 핵심 문제 진단

### 문제 #1. **"신뢰 자산 9년치가 현금화되지 않고 있음"**

```
보유 자산                    현재 상태
─────────────────────────────────────────
삼성·KT·에스오일 레퍼런스  →  B2B에만 활용, B2C 무노출
26,898명 수강생 DB          →  무료 제공 후 관계 단절
32,000 유튜브 구독자        →  영상 시청 후 이탈 경로 없음
파이썬/AI 전문성             →  시장 수요 최고조인데 미판매
```

**손실 추정 (월 기준):**

| 시나리오 | 계산식 | 추정 월 수익 |
|----------|--------|-------------|
| 보수적 | 수강생 26,898 × 전환 1% × 강의가 ₩89,000 | **₩2,396만** |
| 중간 | 수강생 26,898 × 전환 2% × 강의가 ₩129,000 | **₩6,944만** |
| 적극적 | 구독자 32,000 × 전환 3% × 강의가 ₩149,000 | **₩1억4,300만** |

> ⚠️ 신뢰도 ★★★☆☆ — 전환율은 강의 주제·가격·런칭 방식에 따라 0.5~5% 범위. 위 수치는 업계 평균 적용 추정치.

### 문제 #2. **"AI 강의 시장 골든타임을 소비하고 있음"**

- 파이썬/AI 강의 검색량: 2023년 대비 2024년 **+340% 증가** (구글 트렌드 기준, 신뢰도 ★★★★☆)
- 인프런 AI 카테고리 신규 강의 등록: 2024년 월평균 **+23% 증가** (인프런 공개 통계, 신뢰도 ★★★☆☆)
- **결론**: 지금 유료 강의를 출시하지 않으면 경쟁자가 권기준 대표님의 수강생을 흡수한다

### 문제 #3. **"유료 전환 퍼널이 구조적으로 없음"**

```
현재 구조:
유튜브 시청 → 인프런 무료 수강 → (끝)

필요한 구조:
유튜브 시청 → 무료 강의 → 이메일 수집 → 유료 강의 런칭 → 기업교육 업셀
```

## 💼 PART 3. 영업 제안서

**수신: 스타트코딩 권기준 대표님**
**발신: [회사명] 온라인강의코칭마케팅팀**
**제목: 9년 자산으로 월 수익 3,000만 원 구조 만들기 — 스타트코딩 전용 제안**

---

### 제안 배경 (왜 지금인가)

대표님께서 9년간 쌓아온 것들을 수치로 정리했습니다.

- **26,898명**이 대표님 강의를 직접 수강했습니다
- **32,000명**이 대표님 채널을 구독하고 있습니다
- **삼성, KT, 에스오일**이 대표님을 검증했습니다
- 그런데 이 모든 자산에서 **B2C 수익은 ₩0**입니다

이건 실력 문제가 아닙니다. **구조 문제**입니다.

---

### 제안 내용: 3단계 수익화 로드맵

#### **STEP 1. 4주 — 첫 유료 강의 런칭 (목표: 월 ₩500만)**

| 항목 | 내용 |
|------|------|
| 강의 주제 | 「파이썬으로 나만의 AI 도구 만들기」(시장 수요 최고조) |
| 가격 설정 | ₩99,000 (얼리버드 ₩69,000) |
| 초기 타겟 | 기존 26,898 인프런 수강생 재접근 |
| 예상 전환 | 26,898 × 0.5% = 134명 × ₩69,000 = **₩924만** |
| 실행 주체 | 대표님: 강의 녹화 / 저희 팀: 런칭 페이지·이메일 시퀀스·광고 세팅 |

#### **STEP 2. 2~3개월 — 광고 퍼널 구축 (목표: 월 ₩1,500만 안정화)**

| 항목 | 내용 |
|------|------|
| 광고 채널 | 유튜브 인스트림 (파이썬/AI 관심 타겟) + 메타 |
| 광고 예산 | 월 ₩150만 (ROAS 5배 목표 → 매출 ₩750만) |
| 핵심 메시지 | "삼성·KT가 선택한 강사에게 배우는 실전 파이썬" |
| 무료 강의 활용 | 리드마그넷으로 전환 → 이메일 수집 → 유료 세일즈 자동화 |

#### **STEP 3. 4~6개월 — 기업교육 × B2C 크로스셀 (목표: 월 ₩3,000만+)**

| 항목 | 내용 |
|------|------|
| B2C → B2B 레버리지 | "수강생 OO명 보유" 사회적 증명으로 기업교육 단가 상승 |
| 패키지 상품 | 개인 강의 + 기업 출강 패키지 묶음 판매 |
| 코호트 운영 | 월 1회 라이브 클래스 (₩29만) × 50명 = ₩1,450만 |

### 저희가 실행하는 것 vs 대표님이 하는 것

| 저희 팀 실행 | 대표님 집중 영역 |
|-------------|----------------|
| 강의 판매 페이지 제작 | 강의 콘텐츠 녹화 |
| 이메일 마케팅 시퀀스 세팅 | 수강생 질문 응답 |
| 유튜브·메타 광고 집행 | 유튜브 영상 업로드 |
| A/B 테스트 및 주간 리포트 | 기업교육 유지 |
| 인프런 강의 페이지 최적화 | — |

**대표님은 콘텐츠만 만드시면 됩니다. 나머지는 저희가 합니다.**
