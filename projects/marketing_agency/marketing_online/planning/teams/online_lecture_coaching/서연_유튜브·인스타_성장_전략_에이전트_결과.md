## 📍 PART 1. 채널 현황 — 성장 단계 판정

### 현재 위치: **1만→10만 전환기 (단계 2)**

| 판정 항목 | 현황 | 단계 판정 근거 |
|-----------|------|----------------|
| 유튜브 구독자 | 32,000명 | 1만~10만 구간 ✅ |
| 채널 운영 기간 | 9년 | 알고리즘 신뢰도 축적됨 |
| 콘텐츠 수 | 75개 | 월평균 0.7개 → **심각한 저업로드** |
| 수익화 구조 | 0원 | 퍼널 전무 — 즉시 설계 필요 |
| 광고 이력 | 전무 | 유료 채널 완전 미개척 |

**→ 단계 판정: 구독자 수는 2단계이나, 콘텐츠 생산성과 수익화는 0단계 수준**
**→ 전략 방향: 성장보다 수익화 퍼널 先설계 후 성장 가속**

## 📍 PART 2. 유기적 성장 전략 (Organic)

### 🔵 유튜브 SEO · 알고리즘 최적화

#### 현재 문제 진단

```
9년 / 75개 영상 = 월 0.7개 업로드
알고리즘이 원하는 최소 기준: 월 4~8개
→ 알고리즘이 채널을 '휴면 상태'로 인식하고 있을 가능성 높음
```

#### 즉시 실행 — 30일 이내

**[액션 1] 콘텐츠 압축 전략 — 롱폼 1개 → 숏폼 3개 분리**

```
기존 영상 (예: "파이썬 기초 3시간 강의")
↓ 분리
├── 숏폼 #1: "파이썬 변수 10분 완성" (Shorts)
├── 숏폼 #2: "파이썬 반복문 실수 TOP3" (Shorts)  
└── 숏폼 #3: "AI가 파이썬 코드 짜는 법" (Shorts)

→ 기존 콘텐츠 75개 → 숏폼 재활용으로 월 12개 이상 업로드 가능
→ 추가 촬영 없이 업로드 빈도 17배 증가
```

**[액션 2] SEO 키워드 재설계**

| 현재 추정 타겟 키워드 | 추천 키워드로 교체 | 월간 검색량 (추정) |
|----------------------|-------------------|-------------------|
| "파이썬 강의" | "파이썬 독학 순서 2025" | 검색량 상위 / 경쟁도 중 |
| "AI 프로그래밍" | "비전공자 파이썬 취업" | 취업 연계 → 구매 의도 高 |
| "코딩 기초" | "ChatGPT 파이썬 자동화" | 트렌드 키워드 / 급상승 |

> **출처: 유튜브 자동완성 + Google Trends 2025년 1분기 기준 (공개 도구)**

**→ 타이틀 공식: [대상] + [구체적 결과] + [기간/조건]**

```
❌ "파이썬 기초 강의"  
✅ "비전공자가 30일 만에 파이썬으로 자동화 업무 만드는 법"
```

**[액션 3] 썸네일 A/B 테스트 설계**

```
현재 추정 썸네일: 강의형 (교재·코드 화면 중심)
→ 문제: 클릭률(CTR) 유인 요소 부족

변경 공식:
- 배경: 단색 (빨강/파랑/노랑 중 1가지)
- 텍스트: 3단어 이내 (예: "AI 코딩 30분")
- 얼굴: 권기준 대표님 표정 (놀람/확신 중 1가지)
- 수치: 반드시 포함 (30분 / 100% / 0원 등)

목표 CTR: 현재 추정 2~3% → 4~6%로 개선
```

### 🔵 인스타그램 유기적 성장

#### 현재 진단

```
인스타그램 현황: 확인된 공식 계정 없음 (또는 비활성)
→ 인프런 2.6만 수강생의 SNS 유입 채널이 전무한 상태
→ 즉시 개설 또는 활성화 필요
```

#### 90일 인스타 유기적 성장 플랜

**[월 1 — 기반 설계]**

```
계정 바이오 설정:
"삼성·KT 기업교육 강사 | 파이썬·AI 강의 | 수강생 26,898명"
→ 사회적 증명을 바이오에 즉시 노출

콘텐츠 믹스 (주 5회):
- 릴스 3개: 코딩 팁 60초 (예: "파이썬 코드 3줄로 엑셀 자동화")  
- 카드뉴스 2개: "비전공자 파이썬 로드맵" 인포그래픽
```

**[월 2 — 알고리즘 진입]**

```
릴스 훅 공식 (첫 3초가 전부):
"당신이 코딩 못 하는 건 머리가 나빠서가 아닙니다"
→ 감정 자극 → 저장·공유 유도

목표: 릴스 평균 조회수 5,000회 이상 도달
저장률 목표: 6% 이상 (인스타 알고리즘 확산 신호)
```

**[월 3 — 팔로워 → 수강생 전환]**

```
바이오 링크 설계:
인스타 바이오 → 링크트리 → 무료 강의 입문 → 이메일 수집 → 유료 강의 안내

팔로워 목표: 90일 내 1,000명 이상 (유기적)
전환 목표: 팔로워 중 유료 강의 랜딩 클릭률 3% 이상
```

## 📍 PART 3. 유료 광고 전략

> **⚠️ 전제 조건: 유료 강의 출시 전에 광고를 집행하면 예산 낭비입니다. 광고는 반드시 유료 강의 퍼널 완성 후 집행합니다.**

### 광고 집행 순서 (절대 준수)

```
[1단계] 유료 강의 1개 출시 → [2단계] 랜딩페이지 완성
→ [3단계] 무료 → 유료 퍼널 설계 → [4단계] 광고 집행
```

### 🔴 유튜브 광고 전략 (Google Ads)

**광고 상품: 인스트림 광고 (건너뛸 수 없는 6초 / 건너뛸 수 있는 15~30초)**

**타겟 설정:**

```
1차 타겟 (핵심):
- 키워드: "파이썬 독학", "비전공자 코딩", "AI 자동화 배우기"
- 연령: 25~45세
- 관심사: 직무 교육, 커리어 전환, 자기계발

2차 타겟 (리타겟):
- 기존 유튜브 채널 방문자
- 인프런 무료 강의 수강생 (이메일 리스트 활용)
→ 이미 신뢰가 형성된 2.6만 명에게 유료 강의 광고 노출
→ 예상 전환율: 일반 콜드 타겟 대비 3~5배 높음
```

**예산 배분 (월 기준):**

| 광고 유형 | 월 예산 | 예상 노출 | 예상 클릭 | 목표 |
|-----------|---------|-----------|-----------|------|
| 유튜브 인스트림 | 30만원 | 15,000회 | 750회 | 브랜드 인지 |
| 리타겟 (기존 구독자) | 20만원 | 8,000회 | 600회 | 유료 전환 |
| **합계** | **50만원** | **23,000회** | **1,350회** | — |

> **예상 수치 출처: Google Ads 교육 카테고리 평균 CPV 20~30원, CTR 4~6% 적용 (2024~2025 업계 평균)**
