## 2. A/B 테스트 계획 3개

### 테스트 1: 상세페이지 헤드라인 A vs B

**가설**: 빅사이즈 특성을 즉시 인지하는 헤드라인이 검색 적합도 점수와 CTR을 높일 것

**A안(현재)**: "[브랜드] 사이즈명"
```
예: "NIKE 코트비전 280-290mm"
```

**B안(테스트)**: "280~310mm | [브랜드] 사이즈명 | 국내 유일 보유"
```
예: "280~310mm | NIKE 코트비전 | 국내 유일 보유"
```

**측정 지표**: 상세페이지 클릭률(CTR, %)
- 분자: 검색 결과에서 해당 상품 클릭 수
- 분모: 검색 노출 수
- 수집 방법: 네이버 서치어드바이저 > 검색어 성과 > 상품명별 CTR 추이

**테스트 기간**: 14일 (4월 20일~5월 3일)

**테스트 샘플**: 주력 3가지 상품 모델(나이키 코트비전, 아디다스 XXXXX, XXXXX)에 A/B 적용
- 상품 A: A안 유지 (컨트롤)
- 상품 B: B안 적용 (변수)
- 상품 C: C안(하이브리드: "280-310mm | NIKE 코트비전") 적용 (옵션)

**통계적 유의성**: 상품당 최소 클릭수 30회 이상 도달 시 평가
- 목표: B안이 A안 대비 **CTR 0.3%p 이상** 증가 확인

**의사결정**:
- ✅ B안 CTR > A안 CTR + 0.3%p → 모든 상품 헤드라인 B안으로 통일
- ❌ 유의미한 차이 없음 → A안 유지, W4에서 다시 테스트 or 헤드라인 재설계

**가설**: 리뷰 작성 난이도를 낮출 인센티브 구조가 체험단 완료율을 높일 것 (리뷰 0→15개 달성에 필수)

```

체험단 모집 문구: "NIKE 280mm 무료로 착용 후 솔직한 리뷰 작성해주세요. 완료 시 쿠폰 3,000원 증정"
```

**B안**: "제품 무료 제공 + 추가 상품 30% 할인권"
```
구조: 제품 배송 후 리뷰 작성 완료 시 스토어 내 추가 상품 구매 시 30% 할인권 지급
체험단 모집 문구: "NIKE 280mm 무료 + 다음 구매 30% 할인! 솔직한 리뷰로 리워드 받으세요"
```

**측정 지표**: 리뷰 작성 완료율(%)
```
완료율(%) = (리뷰 작성 완료자 / 체험단 배송자) × 100
```

**테스트 기간**: 14일 (4월 22일~5월 5일) — 배송 시간 고려해 W2 중반 시작

**테스트 샘플**: 체험단 총 10명(A안 5명 + B안 5명)
- A안: 나이키 280mm 체험단 5명
- B안: 아디다스 300mm 체험단 5명
- 동일 조건: 배송 후 리뷰 작성 기한 7일로 통일

**통계적 유의성**: 샘플 5명 기준(소규모), 정성 피드백도 병행 수집
- 목표: B안 완료율 > A안 완료율 **+ 15%p 이상**

**의사결정**:
- ✅ B안 완료율 ≥ 80% & A안 완료율 ≤ 60% → B안 채택, W3~W4 추가 체험단 모집 시 B안 인센티브 적용
- ❌ 완료율 차이 ≤ 10%p → 인센티브 구조 재설계 필요 (예: 금전 + 커뮤니티 피드백 병합)
- ⚠️ 두 안 모두 완료율 < 50% → 체험단 모집 채널 변경 필요 (현재 계획된 빅사이즈 커뮤니티 3곳 재검증)
