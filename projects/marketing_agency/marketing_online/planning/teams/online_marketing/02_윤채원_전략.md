## 2. 차별화 포지셔닝 전략 — your_companion 전용

### 왜 "빅사이즈 운동화"는 일반 체험단+SEO 루틴으로 안 되는가

| 일반 루틴의 한계 | your_companion에 필요한 것 |
|-----------------|--------------------------|
| 체험단 리뷰어가 해당 사이즈(280~310mm)를 실제 보유하지 않음 → 착용 후기 작성 불가 | **실제 빅사이즈 착용자**를 리뷰어로 한정 모집 |
| 범용 키워드(운동화, 스니커즈) 경쟁 → 빅파워 셀러에 묻힘 | 니치 롱테일 키워드 집중: "280mm 나이키 코트비전", "310사이즈 아디다스" 등 사이즈+모델명 조합 |
| 상세페이지 개선 = 이미지 추가 → 빅사이즈 특수성 미반영 | 사이즈별 착용샷 + 발볼 너비 실측 표기 + "없던 사이즈를 여기서 찾았다" 스토리 구조 |

### 이 셀러/카테고리에만 통하는 독창적 채널 2개

#### 독창적 채널 A — 빅사이즈 커뮤니티 타겟 바이럴 (스탠다드·프리미엄 공통)

**왜 이 제품에 효과적인가**
빅사이즈(280mm 이상) 착용자는 오프라인 매장에서 재고를 찾지 못하는 경험을 공유하는 커뮤니티(네이버 카페 "키큰남자들의 모임", 디시인사이드 신발 갤러리 빅사이즈 스레드, 인스타그램 #빅사이즈운동화 태그)를 이미 형성하고 있다. 이 커뮤니티에서 "드디어 내 사이즈 찾았다" 형식의 인증 후기 1개가 자생적 공유로 이어지는 구조다. 범용 운동화 광고와 경쟁하지 않고, 수요가 확실한 니치 집단에 직접 도달한다.

**구체적 실행 계획**
- 누가: 윤채원(전략) → 박시우(콘텐츠 제작) → 한소율(커뮤니티 게시 및 반응 모니터링)
- 어디서: 네이버 카페 빅사이즈 관련 카페(직접 조회 필요), 인스타그램 #빅사이즈운동화 태그 계정 DM
- 어떻게: ① 실제 280~310mm 착용자 체험단 3~5명 모집 (사이즈 인증 필수 조건) → ② 착용샷+발볼 실측 후기 작성 → ③ 커뮤니티 자연스러운 정보 공유 형식으로 게시 ("290 신발 못 찾던 내가 여기서 해결함" 구조) → ④ 스토어 URL 자연 삽입
- 스탠다드 미니버전: 체험단 3명 + 인스타 #빅사이즈운동화 태그 게시 1회

#### 독창적 채널 B — 사이즈+모델명 조합 롱테일 SEO (스탠다드·프리미엄 공통)

**왜 이 제품에 효과적인가**
"운동화", "나이키" 같은 범용 키워드는 대형 셀러가 독점하지만, "나이키 코트비전 295", "아디다스 스탠스미스 300사이즈" 같은 사이즈+모델명 조합 키워드는 검색량이 소규모이나 구매 의도가 극히 높다(추정: 해당 키워드 검색자는 이미 모델을 결정하고 사이즈만 찾는 최종 구매 단계 진입자). 46개 상품에 이 구조를 일괄 적용하면 리뷰 없이도 검색 노출 진입 가능성이 상대적으로 높다(추정: 롱테일 키워드 경쟁 상품 수 적음 — 직접 조회 필요).

**구체적 실행 계획**
- 누가: 윤채원(키워드 설계) → 박시우(상품명·상세페이지 카피 적용)
- 어디서: 스마트스토어 상품명 필드 + 상세페이지 텍스트 레이어
- 어떻게: ① 46개 전 상품의 모델명 + 사이즈 범위 키워드 조사(네이버 키워드 플래너 직접 조회 필요) → ② 상품명 구조를 "[브랜드] [모델명] [사이즈범위] 빅사이즈 운동화" 형식으로 재작성 → ③ 상세페이지에 사이즈별 착용 가능 발볼 너비 표 삽입 (구매 결정 장벽 제거) → ④ 2주 후 네이버 쇼핑 검색 순위 변화 모니터링
- 스탠다드 미니버전: 상위 판매 추정 10개 상품 우선 적용

### LEAD-001 | maximaproject (A등급 — 보조)
